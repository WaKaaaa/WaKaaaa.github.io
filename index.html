JavaScript高级程序设计第三版笔记四--操作符

一： 一元操作符

只操作一个值得操作符叫一元操作符（自增，自减，一元加，一元减）

对于非数值的数，++ == 会自动转换类型

  • 执行前置自增和自减，变量的值都是在语句被求值以前改变的

  • 执行后置自增和自减，变量的值都是在语句被求值之后改变的
    ###
    var a = 2;
    var b = ++a;
    alert(a); //3
    alert(b); //3

    var c = 1;
    var d = c++;
    alert(c); //2
    alert(d); //1

    二： 位操作符

    位操作符是按内存中表示的数值的位来操作数值。在ECMAScript中所有数值都是以IEEE-754 的64位格式存储，但是位操作符操作的是32位的数值，所以先将64位转换成32位操作，完成之后再转换回64位

JavaScript高级程序设计第三版笔记三--数据类型

基本数据类型 Undefined Null Boolean Number String 和复杂数据类型 Object，因为ECMAScript数据类型具有动态性，所以这些数据类型就足够了。

一： typeof()检测给定变量的数据类型

  • undefined：如果这个值未定义
  • boolean： 如果这个值是布尔值
  • string： 如果这个值是字符串
  • number： 如果这个值是数值
  • object： 如果这个值是对象或者null
  • function： 如果这个值是函数
    var a = 2;12/20/2016 5:29:31 PM 12/20/2016 5:29:33 PM 12/20/2016 5:29:35 PM
    var d = “”;
    alert(typeof(a)); //number
    alert(typeof null); //object,这里，无论是object还是null，都会返回object类型
    alert(typeof(c)); //undefined
    alert(typeof(d)); //string

    注意1： typeof是一个操作符而不是函数，所以()可以省略也可以写上

    注意2： null被认为是一种空的对象引用，所以返回的是object

    注意3： 在技术层面上讲，函数在ECMAScript中是对象，不是数据类型，然而，函数也确实有一些特殊的属性，因此可以通过typeof操作符来区分函数和其他对象是很有必要的。

    二： undefined类型

    undefined类型只有一个值，就是特殊的undefined。使用var声明变量但没有对其初始化的时候，这个变量的值就是undefined

    注意： 包含undefined的值的变量与尚未定义的变量是不一样的

    var massage;
    alert(massage); //undfined
    alert(a); //产生错误
    alert(typeof massage); //undefined
    alert(typeof a); //undefined

    三： null类型

    null类型只有一个值，null。null值表示一个空指针对象

    var a = null;
    alert(typeof(a));

    undefined和null的区别与联系：

  • undefined 是派生自null的
    ######
    alert(undefined == null); //true
  • 不必把一个undefined的值设置为显示的，但是null就不行
  • 只要你想将某个变量保存对象，但还没有真正的保存任何东西，就应该显示的将该变量值保存为null
    ######

    四： Boolean类型

    Boolean类型的字面值只有true和false两个，但是在ECMAScript中所有类型的值都有与这两个Boolean值等价的值

    Boolean()转型函数可以将一个值转换为对应的Boolean值，可以对任何数据类型调用这个函数Boolean()，而且总会返回一个布尔值（true/false）

    alert(Boolean(true)); //true
    alert(Boolean(false)); //false
    alert(Boolean(“1234”)); //true
    alert(Boolean(“”)); //false
    alert(Boolean(1234)); //true
    alert(Boolean(0)); //false
    alert(Boolean(Infinity)); //true
    alert(Boolean(Array)); //true
    alert(Boolean(null)); //false
    alert(Boolean(undefined)); //false

    五： Number类型

    Number类型存在十进制，八进制，十六进制，在严格模式下是无效的

    ECMAScript可以保存+0和-0，正0和负0被认为相等

    浮点数在计算的时候精确度远远不如整数，所以永远不要测试某个特定的浮点数值，而关于浮点数计算会产生舍入误差的问题是基于IEEE754数值的浮点计算的通病，ECMAScript不是特有的

    数值范围：Infinity -Infinity表示正负无穷

    Number.MIN_VALUE~Number.MAX_VALUE是数值的可取范围，超出之后利用isFinite()可以检测为false

    alert(Number.MAX_VALUE);
    alert(Number.MIN_VALUE);
    var result1 = Number.MAX_VALUE+1;
    var result2 = Number.MIN_VALUE+1;
    alert(isFinite(result1)); //true
    alert(isFinite(result2)); //true

    NaN（not a number），非数值，是一个特殊的数值，用来表示一个本来要返回数值的操作数未返回数值的情况（这样就不会抛出异常了）

    NaN的两个特点

    1. 任何涉及NaN的操作都会返回NaN

    1. NaN与任何数都不相等，包括它本身

      isNaN()函数接受一个任何类型的参数，这个函数会帮助我们确定这个函数是否“不是数值”，这个函数接收一个值后会尝试将这个值转换为数值，使某些不是数值的值会直接转换为数值，而任何不能转换为数值的值都会导致这个函数返回true

      alert(NaN == NaN); //false

      alert(isNaN(‘color’)); //true，不可以转化成数值
      alert(isNaN(false)); //false，会被转化成数值0
      alert(isNaN(true)); //false，会被转化成数值1
      alert(isNaN(NaN)); //true

      注意：isNaN()也适用于对象，在基于对象调用isNaN()函数时，会首先调用对象的valueof()方法，然后确定该方法的值是否可以转换成数值，如果不能，则这个返回值再调用toString()方法，再测试返回值

      数值转换：Number(),perseInt(),perseFloat()这三个函数可以将非数值转换为数值

      number()的转换规则：

  • null — 0
  • undefined — NaN
  • true — 1
  • false — 0
  • 字符串 规则很多
    1. 空字符串 — 0
    1. 只包含数字 — 十进制数字，忽略前导0
    1. 包含有效的浮点数格式 — 对应的浮点数，忽略前导0
    1. 如果包含有效的十六进制 — 相同大小的十进制
    1. 其他的 — NaN

  • 对象 — 会首先调用对象的valueof()方法，然后确定该方法的值是否可以转换成数值，如果不能，则这个返回值再调用toString()方法，再测试返回值
    ######

    parseInt()：会忽略字符串前边的空格，直到找到第一个非空格字符

  • 如果第一个非空格不是数字或者负号 — NaN

  • 空字符串 — NaN(number()是0，这是这两点的区别 )
  • 如果第一个非空格是数字，再解析第二个，直到解析完毕或者遇到非字符 — 显示数字字符
  • parseInt()可以识别各种整数格式（八进制，十进制，十六进制）
    ######
    alert(parseInt(“070”)); //70,这里ECMAScript3和5是不一样的，前者认为是八进制，后者认为是十进制
    // 为了避免出现矛盾，可以给函数添加一个参数，后边的参数代表进制数

    alert(parseInt(“070”,8)); //56

    parseFloat()

  • 字符串中的第一个小数点有效，第二个小数点无效
  • parseFloat()始终会忽略前导0，也就是他只有一个参数，只能解析十进制
    ######

JavaScript高级程序设计第三版笔记一--基础知识

一：JavaScript的实现

1. 一个完整的javascript实现由ECMAScript，DOM，BOM三部分组成。其中

ECMAScript提供核心语言功能

DOM提供访问和操作网页内容的方法和接口

BOM提供与浏览器交互的方法和接口

二：在HTML中使用JavaScript

1. 注意：带有src属性的<script>元素不应该在其<script>和</script>标签之间再包含额外的JavaScript代码。如果包含了嵌入的代码，则只会下载并执行外部脚本文件，嵌入的代码会被忽略。简言之：js代码要么全部写在HTML文件里，要么全部写在单独的文件中

2. 通过src属性可以引入包含外部域的JavaScript文件，但这样做要万分小心，注意程序员对那个文件的恶意修改。如果你想包含来自不同域的代码，要么你是那个域的所有者，要么那个域的所有者值得信赖。

3. 无论如何包含代码，只要不存在defer和async属性，浏览器都会按照<script>元素在页面中出现的先后顺序对他们依次进行解析，也就是说，在第一个script元素包含的代码解析完成之后，第二个script包含的代码才会被解析，然后睡第三个，第四个······

4. 何谓defer和asyn？ defer和async的区别 和defer async的原理及应用

5. 我原先不知道的：在传统的做法中，所有的<script>都应该放在页面的头文件中，这种做法的目的就是将所有的外部文件（CSS和js文件）的引用都放在相同的地方，但这意味着必须等到全部的js代码全部被下载，解析，执行完成之后，才能开始执行页面内容（浏览器遇到body才开始呈现页面内容）。对于那些包含很多js文件的网页来说，这会导致在浏览器呈现页面的时候出现明显的延迟，而延迟期间的浏览器串口是一片空白。为了避免这个问题，现代web应用程序一般都把全部的js引用放在body页面元素中页面的后面。这样，在解析包含的js代码之前，页面的内容将完全的呈现在浏览器中，而用户也会因为浏览器窗口显示空白页面的时间缩短而感到打开页面的速度加快了。

6. 不支持JavaScript的浏览器会将<script>元素的浏览器能够隐藏嵌入的JavaScript代码。解决的方法是把包含JavaScript代码包含在一个HTML注释中。这样，不支持的浏览器就会忽略注释中的代码，支持的浏览器就会进一步确认其中是否包含所需要解析的JavaScript代码

7. <noscript>元素可以指定在不支持脚本的浏览器中显示的替代内容，但是启用了脚本的情况下，浏览器不会显示<noscript>中的任何内容。这个元素支持所有<body>支持的元素，<script>元素除外。

Java算法学习--回溯法--n后问题

一：问题描述

在nn格的棋盘上放着彼此不受攻击的n个皇后，按照国际象棋规定，皇后可以攻击与之处在同一行，同一列，对角线上的皇后。n后问题等价于：在nn的棋盘上，任意两个皇后不在同一行，同一列，同一斜线，当n的值一定时（比如8），有多少种摆放的方法。

算法：

1.递归回溯：

package suanfasheji;

import java.util.Scanner;

public class NQueen1 {
    static int n; // 皇后个数
    static int[] x; // 当前解
    static long sum; //当前已经找到的可行解的方案个数

    public static long nQueen(int nn){
        n = nn;
        sum = 0;
        x = new int[n+1];
        for(int i=0; i<=n; i++) x[i]=0;
        backtrack(1);    //从第一层开始搜索
        return sum;
    }

    private static boolean palce(int k){
        for(int j=1; j<k; j++)
            if(Math.abs(k-j) == Math.abs(x[j]-x[k]) || (x[j]==x[k])) return false;
        return true;
    }

    private static void backtrack(int t){
        if(t>n) sum++;
        else
            for(int i=1; i<=n; i++){
                x[t] = i;
                if(palce(t)) backtrack(t+1);
            }
    }
    public static void main(String[] args) {
        // TODO Auto-generated method stub
        Scanner in = new Scanner(System.in);
        System.out.println("请输入皇后数：");
        int n = in.nextInt();
        NQueen1 qun = new NQueen1();
        long sum = qun.nQueen(n);
        System.out.print(n+"皇后的可行解是："+sum);
    }

}

2.非递归回溯：

package suanfasheji;

import java.util.Scanner;

public class NQueen2 {
    static int n; // 皇后个数
    static int[] x; // 当前解
    static long sum; // 当前已经找到的可行解的方案个数

    public static long nQueen(int nn) {
        n = nn;
        sum = 0;
        x = new int[n + 1];
        for (int i = 0; i <= n; i++)
            x[i] = 0;
        backtrack(); 
        return sum;
    }

    private static boolean palce(int k) {
        for (int j = 1; j < k; j++)
            if (Math.abs(k - j) == Math.abs(x[j] - x[k]) || (x[j] == x[k]))
                return false;
        return true;
    }

    private static void backtrack(){
        x[1] = 0;
        int k=1;
        while(k>0){
            x[k]+=1;
            while((x[k]<=n) && !(palce(k))) x[k]+=1;
            if(x[k]<=n)
                if(k==n) sum++;
                else{
                    k++;
                    x[k]=0;
                }
            else k--;
        }
    }

    public static void main(String[] args) {
        // TODO Auto-generated method stub
        Scanner in = new Scanner(System.in);
        System.out.println("请输入皇后数：");
        int n = in.nextInt();
        NQueen1 qun = new NQueen1();
        long sum = qun.nQueen(n);
        System.out.print(n+"皇后的可行解是："+sum);
    }

}

php--mysql常用函数

总结：

  1. mysql-fetch_array()从结果集中取得一行昨为关联数组，或数字数组，或两者都有

  2. mysql_fetch_row()从结果集中取得一行作为枚举数组

  3. mysql_fetch_assoc()函数以键的方式显示，已经废弃不用了

  4. mysql_field_name()： 取得结果中指定字段的字段名

  5. mysql_num_rows — 取得结果集中行的数目

  6. mysql_num_fields — 取得结果集中字段的数目

详解：

1. mysql-fetch_array()从结果集中取得一行昨为关联数组，或数字数组，或两者都有

$query = 'SELECT * FROM userdetail';
$result = @mysql_query($query) or die('sql语句错误'.mysql_error());

print_r(mysql_fetch_array($result));

mysql_fetch_array()实际上是两个函数(mysql_fetch_row()和mysql_fetch_assoc())的集合，现在这个集合函数用的比较多

2. mysql_fetch_row()从结果集中取得一行作为枚举数组

print_r(mysql_fetch_row($result));

3. mysql_fetch_assoc()函数以键的方式显示，已经废弃不用了

print_r(mysql_fetch_assoc($result));

4. mysql_field_name()： 取得结果中指定字段的字段名，第一个参数：资源句柄，第二个参数是列的下标，从0开始的

echo mysql_field_name($result, 0);        //username

echo mysql_num_rows($result);    //共有6条数据

6. mysql_num_fields — 取得结果集中字段的数目

// 结果集中列的数目
echo mysql_num_fields($result);

7. 取得所有的字段

for($i=0; $i<mysql_num_fields($result); $i++){
    echo mysql_field_name($result, $i);
    echo " ";
}
结果：username userpass regtime role lognum

java算法学习--分治法--合并排序

基本思想： 将待排序的元素分成大致相同的两个子集合，分别对两个子集合进行排序，最终将排好序的子集合合并称为所要求得排好序的集合

算法如下：

public class Mergesort {
// 合并算法
public static void merge(int[] a, int low, int mid, int high) {// 对两组已经排序的数组进行合并，low是第一个元素的下标，high是最后一个元素的下标
    int[] b = new int[high - low + 1]; // 临时数组，存储个数为high - low +
                                        // 1个数据。这个存储的是排好序的元素
    int s = low; // 这里s = 0
    int t = mid + 1; // t = 4+1 = 5
    int k = 0; // k是将要装入b[]中的元素
    // 把较小的数先移到新数组
    while (s <= mid && t <= high) { // 直至前半部或后半部数据完全录入暂存
        if (a[s] <= a[t]) // 如果前半部的第一个数据数据小于后半部的第一个数据，前半部数据暂存如b[]中
            b[k++] = a[s++]; // b[1] = a[1] = 34
        else
            // 否则前半部分的第一个数据大于后半部的第一个数据，后半部第一个数据数据暂存入b[]中，并下标自加
            b[k++] = a[t++];
    }
    // 把左边剩余的数移入数组,也就是右边的已经比较完了
    while (s <= mid)
        // 取元素放入b中
        b[k++] = a[s++];
    // 把右边剩余的数移入数组,right后边的数组已经比较完
    while (t <= high)
        b[k++] = a[t++];
    for (int i = 0; i < b.length; i++) { // 将暂存的数据重新填充至array[low]--array[high]中
        a[low + i] = b[i]; // 将b[]中的元素放入到a[]中
    }
}

// 分割算法
public static void mergesort(int a[], int low, int high) {// 对数组进行递归排序
    int mid;
    if (low < high) { // 至少有两个元素
        mid = (low + high) / 2; // 取中间点
        mergesort(a, low, mid); // 再对左边分割，直至只剩下一个元素
        mergesort(a, mid + 1, high); // 再对右边分割，成为单个元素
        // 已经分割完毕，此时是单个元素
        merge(a, low, mid, high); // 调用 合并两个已经排好序的数组
        System.out.println("low:" + low);
        System.out.println("high:" + high);

    }
}

public static void main(String[] args) {
    int[] a = { 4, 34, 2, 56, 5, 9, 6, 45, 8 };
    System.out.println("排序前数组为：");
    for (int i = 0; i < a.length; i++) {
        System.out.print(a[i] + " ");
        // System.out.println();

    }
    mergesort(a, 0, a.length - 1);
    // System.out.println(a.length); 10

    System.out.println("\n排序后数组为：");
    for (int i = 0; i < a.length; i++) {
        System.out.print(a[i] + " ");
    }
}

}

java算法--分治法--二分搜索技术

问题描述：给定已经排好序的n个元素a[0:n-1]，现在要在这n个元素中搜索一个指定的元素x

问题思路：

  1. 将n个元素分成个数大致相同的两半，取a[n/2]与x进行比较

  2. 如果x = a[n/2]，找到x，算法结束

  3. 如果x < a[n/2]，则在左边继续搜索x

  4. 如果x > a[n/2]，则在右边继续搜索x

算法如下：

public class binary {
    public static int binarySearch(int[] a,int n,int x){  
        //从第0个元素到最后一个
        int left = 0;
        int right = n;
        //找到x所在的位置
        while(left<=right)
        {
            int middle = (left+right)/2;
            if(x == a[middle]) return middle;
            if(x>a[middle]) left = middle+1;
            else right = middle-1;
        }
        return -1;
    }
    public static void main(String[] args){  
        // TODO Auto-generated method stub
        //定义主函数和数组a，调用函数
        //位数是从0开始的。
        int[] a={1,2,3,5,7,9,13,18,24,35,63,89};  
        System.out.println("13在第"+binarySearch(a,11,13)+"位");  
        System.out.println("18在第"+binarySearch(a,11,18)+"位");
        System.out.println("20在第"+binarySearch(a,11,20)+"位");
    }

}
结果：
13在第6位
18在第7位
20在第-1位

算法的时间复杂度是：O(logn)
