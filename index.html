java算法学习--分治法--合并排序

基本思想： 将待排序的元素分成大致相同的两个子集合，分别对两个子集合进行排序，最终将排好序的子集合合并称为所要求得排好序的集合

算法如下：

public class Mergesort {
// 合并算法
public static void merge(int[] a, int low, int mid, int high) {// 对两组已经排序的数组进行合并，low是第一个元素的下标，high是最后一个元素的下标
    int[] b = new int[high - low + 1]; // 临时数组，存储个数为high - low +
                                        // 1个数据。这个存储的是排好序的元素
    int s = low; // 这里s = 0
    int t = mid + 1; // t = 4+1 = 5
    int k = 0; // k是将要装入b[]中的元素
    // 把较小的数先移到新数组
    while (s <= mid && t <= high) { // 直至前半部或后半部数据完全录入暂存
        if (a[s] <= a[t]) // 如果前半部的第一个数据数据小于后半部的第一个数据，前半部数据暂存如b[]中
            b[k++] = a[s++]; // b[1] = a[1] = 34
        else
            // 否则前半部分的第一个数据大于后半部的第一个数据，后半部第一个数据数据暂存入b[]中，并下标自加
            b[k++] = a[t++];
    }
    // 把左边剩余的数移入数组,也就是右边的已经比较完了
    while (s <= mid)
        // 取元素放入b中
        b[k++] = a[s++];
    // 把右边剩余的数移入数组,right后边的数组已经比较完
    while (t <= high)
        b[k++] = a[t++];
    for (int i = 0; i < b.length; i++) { // 将暂存的数据重新填充至array[low]--array[high]中
        a[low + i] = b[i]; // 将b[]中的元素放入到a[]中
    }
}

// 分割算法
public static void mergesort(int a[], int low, int high) {// 对数组进行递归排序
    int mid;
    if (low < high) { // 至少有两个元素
        mid = (low + high) / 2; // 取中间点
        mergesort(a, low, mid); // 再对左边分割，直至只剩下一个元素
        mergesort(a, mid + 1, high); // 再对右边分割，成为单个元素
        // 已经分割完毕，此时是单个元素
        merge(a, low, mid, high); // 调用 合并两个已经排好序的数组
        System.out.println("low:" + low);
        System.out.println("high:" + high);

    }
}

public static void main(String[] args) {
    int[] a = { 4, 34, 2, 56, 5, 9, 6, 45, 8 };
    System.out.println("排序前数组为：");
    for (int i = 0; i < a.length; i++) {
        System.out.print(a[i] + " ");
        // System.out.println();

    }
    mergesort(a, 0, a.length - 1);
    // System.out.println(a.length); 10

    System.out.println("\n排序后数组为：");
    for (int i = 0; i < a.length; i++) {
        System.out.print(a[i] + " ");
    }
}

}

java算法--分治法--二分搜索技术

问题描述：给定已经排好序的n个元素a[0:n-1]，现在要在这n个元素中搜索一个指定的元素x

问题思路：

  1. 将n个元素分成个数大致相同的两半，取a[n/2]与x进行比较

  2. 如果x = a[n/2]，找到x，算法结束

  3. 如果x < a[n/2]，则在左边继续搜索x

  4. 如果x > a[n/2]，则在右边继续搜索x

算法如下：

public class binary {
    public static int binarySearch(int[] a,int n,int x){  
        //从第0个元素到最后一个
        int left = 0;
        int right = n;
        //找到x所在的位置
        while(left<=right)
        {
            int middle = (left+right)/2;
            if(x == a[middle]) return middle;
            if(x>a[middle]) left = middle+1;
            else right = middle-1;
        }
        return -1;
    }
    public static void main(String[] args){  
        // TODO Auto-generated method stub
        //定义主函数和数组a，调用函数
        //位数是从0开始的。
        int[] a={1,2,3,5,7,9,13,18,24,35,63,89};  
        System.out.println("13在第"+binarySearch(a,11,13)+"位");  
        System.out.println("18在第"+binarySearch(a,11,18)+"位");
        System.out.println("20在第"+binarySearch(a,11,20)+"位");
    }

}
结果：
13在第6位
18在第7位
20在第-1位

算法的时间复杂度是：O(logn)

java算法学习--递归与分治策略

1. 分治法的基本思想：将一个规模为n的问题分解成K个规模较小的子问题，这些子问题相互独立且与原问题规模相同。递归地解这些子问题，然后将各个子问题的解合并到原问题的解。

由分治法产生的子问题往往是原问题的较小规模，这就为使用递归技术提供了方便，在这种情况下，反复应用分治手段，可以使子问题与原问题类型一致而其规模不断缩小，最终缩到问题很小可以求解。

在用分治法设计算法时，最好使子问题的规模大致相同，即将一个问题分成大小相等的k个子问题的处理方法是行之有效的。一般设计出的算法是递归算法

3.递归的概念：直接或者间接地调用自身的算法称为递归算法。

4.递归函数：用函数自身给出定义的函数称为递归函数。如阶乘函数，斐波那契数列，排列问题，整数划分问题，汉诺塔问题

java算法学习--动态规划法

1. 动态规划法与分治法类似，基本思想也是将待求解的问题分解成若干个子问题，先求解子问题，然后从子问题中找到原问题的解。

2. 不同之处：适用于动态规划求解的问题，经过分解之后得到的子问题往往是不相互独立的。

3. 如果能够保存已经解决的子问题的答案，在需要的时候再找出已经得到的答案，就可以避免大量重复的计算，从而得到多项式时间算法。

4. 基本思想：为了达到这个目的，可以用一个表记录所有已经解决的子问题的答案.不管是否能用得到，只要他被计算过，就将他填入表中，这就是动态规划的基本思想。

5. 设计动态规划算法的步骤：

  1. 找出最优解的性质，并刻画其结构特性

  2. 递归的定义最优解

  3. 以自底向上的方式计算出最优解

  4. 根据计算最优时得到的信息，构造最优解

java算法学习--分治法 合并排序

合并排序：用分治策略实现对n个元素进行排序的算法。

基本思想：将待排序列元素分成大小大致相同的两个子序列，分别对两个子序列进行排序，最终将排好序的子集合合并成为所要求的排好序的集合。

public class Mergesort {

    public static void merge(int[]a,int low,int mid,int high){//对两组已经排序的数组进行合并
        int[]b=new int[high-low+1]; //临时数组，存储个数为high - low + 1个数据       
        int s=low;
        int t=mid+1;
        int k=0;
        while(s<=mid&&t<=high){   //直至前半部或后半部数据完全录入暂存
            if(a[s]<=a[t])        //如果前半部的数据小于后半部的，前半部数据暂存
                b[k++]=a[s++];
            else                   //否则后半部数据暂存，并下标自加                
                b[k++]=a[t++];
        }
        while(s<=mid)
            b[k++]=a[s++];
        while(t<=high)
            b[k++]=a[t++];
        for(int i=0;i<b.length;i++){     //将暂存的数据重新填充至array[low]--array[high]中
            a[low+i]=b[i];
        }
    }
    public static void mergesort(int a[],int low,int high){//对数组进行递归排序        
        int mid;
        if(low<high){
            mid=(low+high)/2;
            mergesort(a,low,mid);
            mergesort(a,mid+1,high);
            merge(a,low,mid,high);
        }
    }
    public static void main(String[]args){
        int[]a={4,34,2,56,5,9,6,45,8,3};
        System.out.println("排序前数组为：");
        for(int i=0;i<a.length;i++){
            System.out.print(a[i]+" "); }
        mergesort(a,0,a.length-1);
        System.out.println("\n排序后数组为：");
        for(int i=0;i<a.length;i++){
            System.out.print(a[i]+" ");}    
    }

}

java算法--棋盘覆盖

问题描述：在2^k * 2^k的棋盘中有一个特殊的方格，用L型骨牌覆盖除了这个特殊方格之外的其他方格。要求骨牌不能重叠。

用0代表特殊方格，相同的数字代表同一个骨牌，不同骨牌用不同数字表示

算法思路：采用分治法，先将棋盘分成四个小棋盘，那么特殊方格肯定在这四个小棋盘中的一个里边。将没有特殊方格的三个棋盘交界地方覆盖上一个骨牌，使这个骨牌成为特殊方格。再将小棋盘切成4个小小棋盘，寻找没有特殊方格的三个棋盘覆盖，最终全部覆盖完毕。

具体的覆盖方法：

  1. 如果左上棋盘无特殊方格，将左上棋盘的右下角覆盖

  2. 如果左下棋盘无特殊方格，将左下棋盘的右上角覆盖

  3. 如果右上棋盘无特殊方格，将右上棋盘的左下角覆盖

  4. 如果右下棋盘无特殊方格，将右上棋盘的左下角覆盖

图解如下：

代码实现：

public class chessBoard {
// 设置棋盘规格为8
public static int size = 8;
// 设置特殊方格的横坐标（从0开始）为4
public static int dr = 4;
// 设置特殊方格的纵坐标（从0开始）为3
public static int dc = 3;
// 骨牌，从1开始，相同的骨牌用相同的数字，不同的骨牌数字不同，递增。骨牌个数为（8*8-1）/3=21个
public static int tile = 1;
public static int[][] board;


public static void chessBoard(int tr, int tc, int dr, int dc, int size) {
    // 分别是：左上行号，左上列号，特殊方格行号，特殊方格列号，棋盘规格2^k
    if (size == 1)    
        return;
    int t = tile++, // tile是全局变量，表示L型骨牌的编号
    s = size / 2; // 将棋盘分成上左，上右，下左，下右四部分
    // 判断特殊方格是否在左上角的小棋盘中
    if (dr < tr + s && dc < tc + s) {
        // 特殊方格在此盘中，则再次调用自身函数缩小范围
        chessBoard(tr, tc, dr, dc, s);
    } else {
        // 此盘中无特殊方格
        // 用t号L型骨牌覆盖右下角
        board[tr + s - 1][tc + s - 1] = t;    //这里是将t号骨牌覆盖到bord[3][3]的位置上（也就是右下角）
        chessBoard(tr, tc, tr + s - 1, tc + s - 1, s);    //这里特殊方格变成了3,3，将(0,0,3,3,4)这个棋盘分割
    }
    // 右上角（同理）
    if (dr < tr + s && dc >= tc + s) {    //特殊方格在此盘
        chessBoard(tr, tc + s, dr, dc, s);
    } else {
        board[tr + s - 1][tc + s] = t;
        chessBoard(tr, tc + s, tr + s - 1, tc + s, s);
    }
    // 左下角
    if (dr >= tr + s && dc < tc + s) {
        chessBoard(tr + s, tc, dr, dc, s);
    } else {
        board[tr + s][tc + s - 1] = t;
        chessBoard(tr + s, tc, tr + s, tc + s - 1, s);
    }
    // 右下角
    if (dr >= tr + s && dc >= tc + s) {
        chessBoard(tr + s, tc + s, dr, dc, s);
    } else {
        board[tr + s][tc + s] = t;
        chessBoard(tr + s, tc + s, tr + s, tc + s, s);
    }
}

public static void main(String[] args) {
    // 初始化棋盘
    board = new int[size][size];
    // 调用方法
    chessBoard(0, 0, dr, dc, size);
    // 显示棋盘
    for (int[] is : board) {
        System.out.println();
        for (int i : is) {
            System.out.printf("%4d", i);
         }
     }
 }

}

算法的时间复杂度：

设T(k)是算法chessBoard覆盖2^k * 2^k棋盘所需要的时间，从算法的分割策略可知，T(k)满足：

1. k=0, T(k) = O(1)

2. k>1, T(k) = 4T(k-1)+O(1)

解得T(k) = O(4^k)

public class MyMath {
    public static void perm(String[] list , int k,int m)
    {
        //k是第一个数字，m是最后一个数字
        //输出list[k...m]的所有排列方式
        if(k == m) {    
            //只有一个元素.一到递归的出口就输出数组，此数组为全排列
            for(int j=0; j<= m ;j++) {
                System.out.print(list[j]);
            }
            System.out.println();
        }
        else
            //还有很多元素，递归产生排列
        for(int i = k; i <= m ; i++) {
            //for循环将k~m中每个数放到k位置中去，
            MyMath.swap(list, k, i);
            //假设k位置确定，那么对k+1到m的数继续递归
            perm(list, k+1, m); 
            //换过去之后再还原
            MyMath.swap(list, k, i);
        }    
    }

    private static void swap(String[] str, int i, int first) {
        //temp变量实现元素的交换。
         String temp;
         temp = str[first];
         str[first] = str[i];
         str[i] = temp;      
    }

    public static void main(String args[]){ 
        //java中main()方法是Java应用程序的入口方法，JVM通过main方法找到需要启动的运行程序，
        //输入数组
         String[] str = {"a","b","c","d","e","f"};
         //从第一个到最后一个全排列，变换数字可以任意截取其中一部分，比如perm(str,0,2)，
         //只会对前三个数字进行全排列，其余类似
         perm(str, 0, 5);
         //输出list[0..2]的所有排列方式

} 

}
