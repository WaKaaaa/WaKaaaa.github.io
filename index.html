控制器：数据和视图之间的桥梁，数据就是通过控制器将数据发送到视图页面的

创建控制器的方法：

1.手动创建

控制器的位置：app->Http->Controllers下，我们就是在那里创建有意义的控制器

第一步：在controller下新建一个文件，取一个有意义的名字，例如：InputController

第二步：填写内容

<?php 
namespace App\Http\Controllers;    //指定命名空间，在它下边

class InputController extends Controller
{
    public function input()    //创建一个函数
    {
        echo "这是我手动创建的控制器";
    }

}
// 注意：laravel里所有用浏览器访问的函数都需要到routes里配置路径
// 所以要回到路由器配置一下路径app,http,routes

第三步：去routes里配置，用路由取访问控制器

// 定义指向控制器路由的动作，user/{id}是索要访问的页面，UserCotroller是你定义的控制器的名称，showProfile是UserCotroller里边的方法
// Route::get('user/{id}', 'UserController@showProfile')
Route::get('test','InputController@input');

第四步：输入localhost:8000/test，显示这是我手动创建的控制器

开启laravel的方法：cd到Laravel目录，命令行输入 php artisan serve

PHP Warning: Module ‘openssl’ already loaded in Unknown on line 0的解决方法：php.ini中搜索openssl，有两个相同的 ;extension=php_openssl.dll，将其中一个的分号去掉，一个保留

php artisan make:composer不能找到命名空间。composer语法错误，有可能是conposer.json文件有注释修改等不合语法的操作

2.命令行创建

第一步：在laravel的目录下打开命令行，快捷键是进入目录，shift+右键打开

第二步：输入：php artisan make:controller 你的控制器名字，比如php artisan make:controller HelloController,就可以创建一个HelloController的控制器，配置如下

<?php

    }

php 二维数组

##1.定义

数组的元素本身仍然是数组的数组成为二维数组，可以把二维数组当成一个矩阵，或者是一个具有宽度和高度或者行和列的网格。一维数组只在一行或者一列上显示，二维数组则像网格一样有好几行和好几列。

 ?>

3.打印内外数组长度的方法

$students = array(
    array('aaa',18,'man'), //第0行
    array('bbb',19,'woman'), //第1行
    array('ccc',17,'man')    //第2行
    );
echo count($students);    //打印外数组长度的方法，即student的长度，是里边的三个数组，结果是3
echo count($students[1]);    //3
echo count($students[2]);    //3
echo count($students[0]);    //打印内数组的长度的方法，即student的第1个元素的长度，结果是3

4.打印数组元素的方法

没有键的情况

$students = array(
array('aaa',18,'man',), //第0行
array('bbb',19,'woman','green'), //第1行
array('ccc',17,'man','red')    //第二行
);
// 这两种方法适合下标为0,1,2...的情况
//打印二维数组的第一种方法，此方法很麻烦
echo $students[0][0]."|".$students[0][1]."|".$students[0][2]."<br>";
echo $students[1][0]."|".$students[1][1]."|".$students[1][2]."<br>";
echo $students[2][0]."|".$students[2][1]."|".$students[2][2]."<br>";

//打印数组的第二种方法：for循环,两重循环打印二维数组
for ($i=0; $i < count($students); $i++) { 
    // echo $students;    //此时打印3个array，这是求student含有几个数组
    for ($j=0; $j < count($students[$i]); $j++) {     //这里用$i的原因：就算内层数组里元素的个数各不相同也可以打印的出来
        echo $students[$i][$j]."|";
    }
    echo "<br>";    //外层循环3次，每次加1个换行
}
结果：

//第二种方法：list和each
    for ($i=0; $i <count($students); $i++) {

            echo $key.'--'.$value.'|';

 ?>

    for ($i=0; $i < count($userName); $i++) { 
    # code...
echo $i."---".$userName[$i]."<br>";    //count方法取得数组的长度，sizeof是他的别名，一般用count
    }
    结果是： 0—张三
    1---李四
2---王五

//这种方法好处是在数组的下标不是从0开始或者根本不是数字的时候可用
foreach ($userName as $key => $value) { //此处的key和value可以替换成任意的变量名，只需要将下面输出统一即可，也可以单独打印下标或者值
    # code...
    echo $key."---".$value."<br>";
}

       0---张三
       1---李四
       2---王五
       3---赵六

3.each 遍历数组

<?php 
    $userAges = array('张三' => 22, '李四' => 21, '王五'=>18);
    // print_r($userAges);
    //each的使用
    //each --  返回数组中当前的键／值对并将数组指针向前移动一步 
    //这里有一个指针，默认情况下，指针是指向第一个键值对
    //这里的第一个键值对是 ,张三=>22
    //如果each($userAges)，那么获取的就是第一个键值对 '张三'=>22
    //each这个函数返回的是一个数组，
    //each将第一个键值对获取到，然后包装成一个新数组。

    each($userAges); //这是第一步，将张三，22取出来，包装成一个新的数组
    $a = each($userAges);    //这是第二步，将李四，21取出
    print_r($a)."<br>";
    echo $a[0];
    // 第一步的结果：Array ( [1] => 22 [value] => 22 [0] => 张三 [key] => 张三 ) 张三
    // 第二步的结果：Array ( [1] => 21 [value] => 21 [0] => 李四 [key] => 李四 ) 李四

 ?>

each循环打印

while (!!$a = each($userAges)) { //当是真，即括号里的值是1的时候执行以下语
    # code...
    echo $a['key']."---".$a['value']."<br>";
}

！！的作用：将真实存在的数值转换成布尔值，在这里each($userAges)返回的是array，即数组里有数据，有数据，用！！可以将这个array转化成1

4.list 遍历数组

<?php 
    $userAges = array("张三","李四","王五");
    // print_r($userAges);

    list($a,$b) = $userAges;    //这里将数组userAges的第一个值赋给$a,第二个值赋给$b，其中$a,$b的值可以改编为任意的变量
    echo $a."<br>";
    echo $b."<br>";

    // 注意：list只能识别下标是0,1,2等数字的数组，自定义的key不能识别的
    // 如果一定要打印数组的键和值，就用each和list的组合
    $userAge = array('张三' => 22, '李四' => 21, '王五'=>18);

    echo $c;    // 打印张三

1.数组命名规则

开头是$，后面跟字母或者下划线，其后的为字母，数字，下划线，不可以有其他元素

$1fu = array('1','2');    //不可以数字开头
$fu_&hello = array("hello","world");    //含有特殊符号&
$userName = array('张三','李四','王五','赵六','嘻嘻嘻','嘿嘿嘿'); //正确的命名

注意：在一个程序中，标量变量和数组变量不能同名，例如已经存在$userName的变量，再创建$userName的数组，后者会覆盖前者

数组变量的名称是区分大小写的，$user和$User是不同的两个变量数组类名

PHP常量和变量都区分大小写

<?php 
    $a = "hello";

    echo $a."<br>";
    echo $A."<br>";

    define("c", 123);

    echo c."<br>";
    echo C;
    // 都打印，常量也区分大小写

 ?>
