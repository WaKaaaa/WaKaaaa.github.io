六：mktime()返回指定日期的时间戳

格式：mktime(时，分，秒，月，日，年);

$time = mktime(8,8,8,8,8,2008);
echo date('Y-m-d H:i:s',$time);

七：strtotime()将人们可读的日期转化为时间戳（和getdate()相反），用这个函数的频率高

用它可计算时间差

$time = strtotime('2016-11-13 20:20:20');
$now = time();
// echo time();

echo getlastmod();    //返回时间戳

php学习--日期与时间

应用场合：表单提交时间，用户登录时间，数据库中数据的更新和删除的时间等

总结：

time()：获取当前时间戳（但是这个比较麻烦，参数含义不明确，一般不使用这个）

getdate()：将时间戳转化为人们可以看得懂的时间，数组形式

strtotime()将人们可读的日期转化为时间戳

mktime()将时间转化成时间戳

getlastmod()返回当前文件最后修改时间的时间戳

checkdate()：检验日期是否合法，boolean形式

date()：格式化本地时间或者日期，可以打印当前时间，date(‘Y-m-d h:i:sa’)

gettimeofday()获取当前时间，数组形式

localtime()返回本地时间（不常用）

date_default_timezone_get()获取默认时区

date_default_timezone_set()修改时区

一：checkdate()函数检测日期是否合法，布尔型

二：date()格式化一个本地时间或日期

格式：string date ( string $format [, int $timestamp ] )

date()的两个参数分别表示一个日期（必选）和一个时间戳（不写的话默认为Unix时间戳）。

date()表示的年月日时分秒

规则：

Y表示四位数的年份，y表示两位数的年份
M表示英文的月份缩写，m表示阿拉伯数字的月份
D表示英文的日缩写，d表示阿拉伯的日

H表示24小时的小时，h表示12小时的计时法
i表示分钟
s表示秒

a是小写的上午和下午

date()的第一个参数可以放一些format目录里不存在的符号，他们不会被解析，看起来排版清晰

打印目前的时间

echo date('Y-m-d h:i:sa');    //返回的是Unix时间戳的时间2016-11-13 08:34:23am，是东八区的时间，可能和实际时间不相同。

时间戳的概念：时间戳是文件属性中的创建，修改和访问时间。

原理：通过其他加密法将时间的数值转化为加密的数值，加密位数会发生改变安

好处：防止数值窃取之后被非法利用，起到了加密的作用

Unix时间戳：从1970年一月一日零时零分零秒到当前时刻，他是一个时间差，与时区没有关系

扩展： microtime() 返回当前 Unix 时间戳和微秒数，也就是gettimeofday()的前两个

作用：计算脚本运行时间（指的是很小很小的时间的差）

function fn(){
    list($a,$b) = explode(' ', microtime());    //以空格分割字符串，将分割后的字符串分别赋值给$a,$b
    return $a+$b;
}
$start = fn();
for ($i=0; $i<100000000 ; $i++) {     //时间越大，运行时间越大
    # code...
}
$end = fn();

echo round($end - $start,4);

运行此脚本的时间是3.1149秒

laravel学习--控制器

控制器：数据和视图之间的桥梁，数据就是通过控制器将数据发送到视图页面的

创建控制器的方法：

1.手动创建

控制器的位置：app->Http->Controllers下，我们就是在那里创建有意义的控制器

第一步：在controller下新建一个文件，取一个有意义的名字，例如：InputController

第二步：填写内容

<?php 
namespace App\Http\Controllers;    //指定命名空间，在它下边

class InputController extends Controller
{
    public function input()    //创建一个函数
    {
        echo "这是我手动创建的控制器";
    }

}
// 注意：laravel里所有用浏览器访问的函数都需要到routes里配置路径
// 所以要回到路由器配置一下路径app,http,routes

第三步：去routes里配置，用路由取访问控制器

// 定义指向控制器路由的动作，user/{id}是索要访问的页面，UserCotroller是你定义的控制器的名称，showProfile是UserCotroller里边的方法
// Route::get('user/{id}', 'UserController@showProfile')
Route::get('test','InputController@input');

第四步：输入localhost:8000/test，显示这是我手动创建的控制器

开启laravel的方法：cd到Laravel目录，命令行输入 php artisan serve

PHP Warning: Module ‘openssl’ already loaded in Unknown on line 0的解决方法：php.ini中搜索openssl，有两个相同的 ;extension=php_openssl.dll，将其中一个的分号去掉，一个保留

####
;;;;;;;;;;;;;;;;;;;
; Module Settings ;
;;;;;;;;;;;;;;;;;;;
里的去掉分号，另一个保留

php artisan make:composer不能找到命名空间。composer语法错误，有可能是conposer.json文件有注释修改等不合语法的操作

2.命令行创建

第一步：在laravel的目录下打开命令行，快捷键是进入目录，shift+右键打开

第二步：输入：php artisan make:controller 你的控制器名字，比如php artisan make:controller HelloController,就可以创建一个HelloController的控制器，配置如下

<?php

namespace App\Http\Controllers;

use Illuminate\Http\Request;

use App\Http\Requests;

class HelloController extends Controller
{

    }
}

PHP学习--数学函数

1.数字相加

主要处理整型和浮点型，如果计算中包含数值和字符串，字符串会在完成计算之前转换为数值，而数值则会在与字符串连接之前转换为字符串。

$a = '5'; //完成计算之前转化为数字
$b = $a+7;
echo "$b";
结果：12

$z = 'a';
echo $z+7;    //因为字母没法办法转换成数字，所以输出7

PHP提供了大量函数来检查变量的数据类型

is_numeric()检验是否为数字或者数字字符串，如果是返回1，如果不是，返回空

$a = '5';    
echo is_numeric($a);
echo "<br>";
结果：1

is_int()检测是否为整型,如果是返回1，如果不是，返回空

$e = 5;
echo is_int($e);    //返回1

is_float()检测是否为浮点型,如果是返回1，如果不是，返回空

$a = 6.1;
$b = 6;
echo is_float($a);    //返回1
echo is_float($b);    //返回空

2.产生随机数

rand()函数是libc(构建PHP所用编译器提供的基本库之一)中定义的随机函数的一个简单包装器。mt_rand()是一个很好的替代实现，提供了很多精心设计的特性，而且mt_rand()甚至比libc中的版本还要快

echo rand(1,100);    //产生随机数，第一个参数是最小值，第二个参数是最大值，自定义的，可改变可不写
echo "<br>";
echo mt_rand(1,100);    //产生更好的随机数，速度比rand快

echo getrandmax();    //获取rand的最大值 4132767
echo "<br>";
echo mt_getrandmax();    //获取mt_rand的最大值 2147483647
echo "<br>";

3.格式化数据

number_format()函数可以把整数和浮点数值转换为一种可读的字符串表示，格式化数字字符串，里边只能放浮点型

//格式化字符串number_format
$f = 123892.4567;
echo number_format($f,3);        //第一位存放浮点数，第二位表示精确到小数点后n位
echo "<br>";
echo number_format($f);    //只有浮点数默认会舍弃小数部分，整数部分按照外国人的习惯每三位用有个逗号隔开
echo "<br>";
echo number_format($f,3,'%',"#");    //第三位将分隔符逗号改成%，第四位将小数点点号改为#号，一般情况下不用这两个
echo "<br>";
结果：123,892.457   123,892   123#892%457
