php 二维数组

##1.定义

数组的元素本身仍然是数组的数组成为二维数组，可以把二维数组当成一个矩阵，或者是一个具有宽度和高度或者行和列的网格。一维数组只在一行或者一列上显示，二维数组则像网格一样有好几行和好几列。

3.打印内外数组长度的方法

$students = array(
    array('aaa',18,'man'), //第0行
    array('bbb',19,'woman'), //第1行
    array('ccc',17,'man')    //第2行
    );
echo count($students);    //打印外数组长度的方法，即student的长度，是里边的三个数组，结果是3
echo count($students[1]);    //3
echo count($students[2]);    //3
echo count($students[0]);    //打印内数组的长度的方法，即student的第1个元素的长度，结果是3

4.打印数组元素的方法

没有键的情况

$students = array(
array('aaa',18,'man',), //第0行
array('bbb',19,'woman','green'), //第1行
array('ccc',17,'man','red')    //第二行
);
// 这两种方法适合下标为0,1,2...的情况
//打印二维数组的第一种方法，此方法很麻烦
echo $students[0][0]."|".$students[0][1]."|".$students[0][2]."<br>";
echo $students[1][0]."|".$students[1][1]."|".$students[1][2]."<br>";
echo $students[2][0]."|".$students[2][1]."|".$students[2][2]."<br>";

//打印数组的第二种方法：for循环,两重循环打印二维数组
for ($i=0; $i < count($students); $i++) { 
    // echo $students;    //此时打印3个array，这是求student含有几个数组
    for ($j=0; $j < count($students[$i]); $j++) {     //这里用$i的原因：就算内层数组里元素的个数各不相同也可以打印的出来
        echo $students[$i][$j]."|";
    }
    echo "<br>";    //外层循环3次，每次加1个换行
}

 ?>

3.each 遍历数组

<?php 
    $userAges = array('张三' => 22, '李四' => 21, '王五'=>18);
    // print_r($userAges);
    //each的使用
    //each --  返回数组中当前的键／值对并将数组指针向前移动一步 
    //这里有一个指针，默认情况下，指针是指向第一个键值对
    //这里的第一个键值对是 ,张三=>22
    //如果each($userAges)，那么获取的就是第一个键值对 '张三'=>22
    //each这个函数返回的是一个数组，
    //each将第一个键值对获取到，然后包装成一个新数组。

    each($userAges); //这是第一步，将张三，22取出来，包装成一个新的数组
    $a = each($userAges);    //这是第二步，将李四，21取出
    print_r($a)."<br>";
    echo $a[0];
    // 第一步的结果：Array ( [1] => 22 [value] => 22 [0] => 张三 [key] => 张三 ) 张三
    // 第二步的结果：Array ( [1] => 21 [value] => 21 [0] => 李四 [key] => 李四 ) 李四

 ?>

each循环打印

while (!!$a = each($userAges)) { //当是真，即括号里的值是1的时候执行以下语

}

！！的作用：将真实存在的数值转换成布尔值，在这里each($userAges)返回的是array，即数组里有数据，有数据，用！！可以将这个array转化成1

4.list 遍历数组

<?php 
    $userAges = array("张三","李四","王五");
    // print_r($userAges);
    // 把一个数组中的值赋给一个变量
    list($a,$b) = $userAges;    //这里将数组userAges的第一个值赋给$a,第二个值赋给$b，其中$a,$b的值可以改编为任意的变量
    echo $a."<br>";
    echo $b."<br>";

    // 注意：list只能识别下标是0,1,2等数字的数组，自定义的key不能识别的
    // 如果一定要打印数组的键和值，就用each和list的组合
    $userAge = array('张三' => 22, '李四' => 21, '王五'=>18);
    list($c,$d) = each($userAge);    // each将张三，22取出来形成一个新的数组，0($c)表示key，也就是张三，1($d)表示value，也就是 22
    echo $c;    // 打印张三
    echo $d;    // 打印22
 ?>

1.数字相加

主要处理整型和浮点型，如果计算中包含数值和字符串，字符串会在完成计算之前转换为数值，而数值则会在与字符串连接之前转换为字符串。

$a = '5'; //完成计算之前转化为数字
$b = $a+7;
echo "$b";
结果：12

$z = 'a';
echo $z+7;    //因为字母没法办法转换成数字，所以输出7

PHP提供了大量函数来检查变量的数据类型

is_numeric()检验是否为数字或者数字字符串，如果是返回1，如果不是，返回空

$a = '5';    
echo is_numeric($a);
echo "<br>";
结果：1

is_int()检测是否为整型,如果是返回1，如果不是，返回空

$e = 5;
echo is_int($e);    //返回1

is_float()检测是否为浮点型,如果是返回1，如果不是，返回空

$a = 6.1;
$b = 6;
echo is_float($a);    //返回1
echo is_float($b);    //返回空

2.产生随机数

rand()函数是libc(构建PHP所用编译器提供的基本库之一)中定义的随机函数的一个简单包装器。mt_rand()是一个很好的替代实现，提供了很多精心设计的特性，而且mt_rand()甚至比libc中的版本还要快

echo rand(1,100);    //产生随机数，第一个参数是最小值，第二个参数是最大值，自定义的，可改变可不写
echo "<br>";
echo mt_rand(1,100);    //产生更好的随机数，速度比rand快

echo getrandmax();    //获取rand的最大值 4132767
echo "<br>";
echo mt_getrandmax();    //获取mt_rand的最大值 2147483647
echo "<br>";

3.格式化数据

number_format()函数可以把整数和浮点数值转换为一种可读的字符串表示，格式化数字字符串，里边只能放浮点型

//格式化字符串number_format
$f = 123892.4567;
echo number_format($f,3);        //第一位存放浮点数，第二位表示精确到小数点后n位
echo "<br>";
echo number_format($f);    //只有浮点数默认会舍弃小数部分，整数部分按照外国人的习惯每三位用有个逗号隔开
echo "<br>";
echo number_format($f,3,'%',"#");    //第三位将分隔符逗号改成%，第四位将小数点点号改为#号，一般情况下不用这两个
echo "<br>";
结果：123,892.457   123,892   123#892%457
