php--mysql常用函数

  view

总结：

  1. mysql-fetch_array()从结果集中取得一行昨为关联数组，或数字数组，或两者都有

  2. mysql_fetch_row()从结果集中取得一行作为枚举数组

  3. mysql_fetch_assoc()函数以键的方式显示，已经废弃不用了

  4. mysql_field_name()： 取得结果中指定字段的字段名

  5. mysql_num_rows — 取得结果集中行的数目

  6. mysql_num_fields — 取得结果集中字段的数目

详解：

1. mysql-fetch_array()从结果集中取得一行昨为关联数组，或数字数组，或两者都有

$query = 'SELECT * FROM userdetail';
$result = @mysql_query($query) or die('sql语句错误'.mysql_error());

print_r(mysql_fetch_array($result));

结果：

Array
(
    [0] => 123
    [username] => 123
    [1] => 123456
    [userpass] => 123456
    [2] => 2016-11-23 21:15:14
    [regtime] => 2016-11-23 21:15:14
    [3] => 0
    [role] => 0
    [4] => 0
    [lognum] => 0
)

mysql_fetch_array()实际上是两个函数(mysql_fetch_row()和mysql_fetch_assoc())的集合，现在这个集合函数用的比较多

2. mysql_fetch_row()从结果集中取得一行作为枚举数组

print_r(mysql_fetch_row($result));

3. mysql_fetch_assoc()函数以键的方式显示，已经废弃不用了

print_r(mysql_fetch_assoc($result));

4. mysql_field_name()： 取得结果中指定字段的字段名，第一个参数：资源句柄，第二个参数是列的下标，从0开始的

echo mysql_field_name($result, 0);        //username
echo mysql_field_name($result, 1);        //userpass

5. mysql_num_rows — 取得结果集中行的数目

echo mysql_num_rows($result);    //共有6条数据

6. mysql_num_fields — 取得结果集中字段的数目

// 结果集中列的数目
echo mysql_num_fields($result);

7. 取得所有的字段

for($i=0; $i<mysql_num_fields($result); $i++){
    echo mysql_field_name($result, $i);
    echo " ";
}
结果：username userpass regtime role lognum

8.

mysql_get_client_info()： 取得 MySQL 客户端信息
mysql_get_host_info()： 取得 MySQL 主机信息
mysql_get_proto_info()： 取得 MySQL 协议信息
mysql_get_server_info()： 取得 MySQL 服务器信息
