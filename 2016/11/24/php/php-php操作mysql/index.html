php学习--php操作mysql

  view

第一步： 连接mysql服务器

第一个参数是服务器地址，第二个参数是服务器的用户名，第三个参数是服务器密码

header('content_type:text/html; charset = utf8');
mysql_connect('localhost:3306 ','root','root');
// 加上@后就算有错误也不会报错，但是最好不要这样
// @mysql_connect('localhost:3306 ','root','root');

更好的连接数据库的方法： 在商业的代码中，一般将这三个参数设置为常量，这样不易修改，就会更加安全

define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PWD', 'root');
define('DB_NAME', 'fu');
$conn = @mysql_connect(DB_HOST,DB_USER,DB_PWD) or die('数据库连接失败，错误信息是：'.mysql_error());

mysql_error返回上一个mysql操作产生的文本错误信息,如果不指定连接资源号，就使用上一个成功打开的连接从mysql服务器提取错误信息,这里就没有指定连接资源号

die语法等同于exit()

第二步：选择数据库

// 设置字符集
mysql_select_db(DB_NAME) or die("表错误，错误信息".mysql_error());
mysql_query('SET NAMES UTF8') or die('字符集设置错误'.mysql_error());

第三步：从这个数据库中选择一张数据表，然后将数据表的信息提取出来（获取记录集）

// mysql_query()发送一条mysql查询，返回资源句柄
$query = 'select * from userdetail';
$result = mysql_query($query) or die("查询错误，错误信息".mysql_error());

// 将记录集里的数据显示出来
// mysql-fetch_array()从结果集中取得一行昨为关联数组，或数字数组，或两者都有
// 以属性--值的方式打印
print_r(mysql_fetch_array($result,MYSQL_ASSOC));
// 以数字下标打印
print_r(mysql_fetch_array($result,MYSQL_NUM));

第四步： 释放资源

mysql_free_result($result);

第五步： 关闭数据库

mysql_close($conn);
