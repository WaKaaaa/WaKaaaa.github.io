php学习--上传文件

  view

总结：

1. enctype = “multipart/form-data” 这句话写在form元素中，是固定写法，没有的话将无法上传文件

2. $_FILES 接受上传的文件，返回一个数组

3. FILE是魔法常量，将绝对地址打印出来

4. dirname(FILE)将文件名的上一级路径打印出来（也就是不打印文件名）

5. // is_dir判断给定的目录是否存在

6. mkdir(name,0777)自动创建一个目录

7. is_uploaded_file()判断文件是否通过http post 上传的

8. move_uploaded_file(文件临时路径，新路径)将上传的文件移动到新位置

详解：

1.

basic1.php

<form action="basic2.php" method = "post" enctype = "multipart/form-data">
    <input type="hidden" name = "MAX_FILE_SIZE" value = "1000000"/>
    ......    
</form>

enctype = “multipart/form-data” 这句话写在form元素中，是固定写法，没有的话将无法上传文件

隐藏域定义了一个问价件大小的限制，必须放在上传域之前，对浏览器的建议，不能超过PHP_ini中的值

2.

basic2.php

print_r($_FILES);

这个函数接受到一个上传的文件，返回内容如下：

Array
(
    [userfile] => Array
        (
            [name] => 4.jpg          //上传文件的文件名
            [type] => image/jpeg        //文件类型
            [tmp_name] =>D:\xampp1\tmp\php26AB.tmp             
                                        //上传的文件临时的存放位置
            [error] => 0                //2是文件超出了HTML设置的大小，
                                       4是空文件 
                                        3是上传了一半 
                                       1是超过了配置文件的大小
            [size] => 7960            //上传文件的大小
 )

3. 限制上传文件的格式为png,jpeg,gif

$Max = array('image/png','image/jpeg','image/gif');
if (is_array($Max)) {
    if(!in_array($_FILES['userfile']['type'], $Max)){
        echo "<script> alert('只允许上传jpg，png格式的文件！'); history.back();</script>";
    }
}

4. 上传错误的提示信息

if($_FILES['userfile']['error'] > 0){
    switch ($_FILES['userfile']['error']) {
        case 1: echo "<script>alert('上传文件超过约定值'); history.back();</script>";
            break;
        case 2: echo "<script>alert('上传文件超过约定值'); history.back();</script>";
            break;
        case 3: echo "<script>alert('文件未全部上传'); history.back();</script>";
            break;
        case 4: echo "<script>alert('请选择上传的文件'); history.back();</script>";
            break;

        default: echo "<script>alert('未知错误，请重新上传'); histor.back();</script>";
            break;
    }
    exit;
}

5.

if (is_uploaded_file($_FILES['userfile']['tmp_name'])) {
    // 如果文件已经上传了，就在这移动
    // 第一个参数，临时文件的地址
    // 第二个参数，要移动的地址
    if(!move_uploaded_file($_FILES['userfile']['tmp_name'], URL."/".$_FILES['userfile']['name'])){
        echo "<script>alert('移动失败'); history.back();</script>";
        exit;
    }
}else{
    echo "<script>alert('临时文件夹找不到文件'); history.back();</script>";
    exit;
}


// 如果全部通过就上传成功了
// 将文件交给一个上传地址
echo "<script>alert('文件上传成功'); location.href = 'basic3.php?url = ".$_FILES['userfile']['name']."';</script>";
