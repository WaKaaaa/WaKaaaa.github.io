php学习--简单的随机图像验证码

  view

完整代码：

ini_set("error_reporting", "E_ALL & ~E_NOTICE");
for ($i=0;$i<4;$i++) {
    $nmsg .= dechex(mt_rand(0,15));
}

header('Content-Type: image/png');
$im = imagecreatetruecolor(75,25);
$blue = imagecolorallocate($im,0,102,255);
$white = imagecolorallocate($im,255,255,255);
imagefill($im,0,0,$blue);
imagestring($im,5,20,5,$nmsg,$white);    //将四位验证码加入到图片中去
imagepng($im);
imagedestroy($im);

解析：

思路：首先生成四位随机的数字字母，这里可以用十六进制数字表示，然后将这四位随机数放在我们已经写好的图片里即可

生成四位数字

for ($i=0;$i<4;$i++) {
    $nmsg .= dechex(mt_rand(0,15));
}
// mt_rand(范围可选) 生成一个更好的随机数
// dechex()将十进制的随机数转化成十六进制

在这里会产生一个报错：Notice: Undefined variable: nmsg in D:\xampp1\htdocs\PHP\Basic14\demo2.php on line 4，修改的方法：

1. 修改php.ini配置文件，将error_reporting = E_ALL修改为error_reporting = E_ALL &~E_NORICE,原理是减弱了PHP的敏感程度，再次遇到这种问题不会报错

2. 在PHP头部加入代码：ini_set("error_reporting", "E_ALL & ~E_NOTICE");，也可以解决这类问题
