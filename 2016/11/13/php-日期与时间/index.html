php学习--日期与时间

  view

应用场合：表单提交时间，用户登录时间，数据库中数据的更新和删除的时间等

总结：

time()：获取当前时间戳（但是这个比较麻烦，参数含义不明确，一般不使用这个）

getdate()：将时间戳转化为人们可以看得懂的时间，数组形式

strtotime()将人们可读的日期转化为时间戳

mktime()将时间转化成时间戳

getlastmod()返回当前文件最后修改时间的时间戳

checkdate()：检验日期是否合法，boolean形式

date()：格式化本地时间或者日期，可以打印当前时间，date(‘Y-m-d h:i:sa’)

gettimeofday()获取当前时间，数组形式

localtime()返回本地时间（不常用）

date_default_timezone_get()获取默认时区

date_default_timezone_set()修改时区

一：checkdate()函数检测日期是否合法，布尔型

二：date()格式化一个本地时间或日期

格式：string date ( string $format [, int $timestamp ] )

date()的两个参数分别表示一个日期（必选）和一个时间戳（不写的话默认为Unix时间戳）。

date()表示的年月日时分秒

规则：

Y表示四位数的年份，y表示两位数的年份
M表示英文的月份缩写，m表示阿拉伯数字的月份
D表示英文的日缩写，d表示阿拉伯的日

H表示24小时的小时，h表示12小时的计时法
i表示分钟
s表示秒

a是小写的上午和下午

date()的第一个参数可以放一些format目录里不存在的符号，他们不会被解析，看起来排版清晰

打印目前的时间

echo date('Y-m-d h:i:sa');    //返回的是Unix时间戳的时间2016-11-13 08:34:23am，是东八区的时间，可能和实际时间不相同。

时间戳的概念：时间戳是文件属性中的创建，修改和访问时间。

原理：通过其他加密法将时间的数值转化为加密的数值，加密位数会发生改变安

好处：防止数值窃取之后被非法利用，起到了加密的作用

Unix时间戳：从1970年一月一日零时零分零秒到当前时刻，他是一个时间差，与时区没有关系

三：gettimeofday()获取当前时间，返回一个关联数组，包含有系统调用返回的数据

###
print_r(gettimeofday()); //数组类型的建议都用print_r()打印详细信息
结果是：
Array
(
[sec] => 1479027519
[usec] => 540985
[minuteswest] => -60
[dsttime] => 0
)
“sec” - 自 Unix 纪元起的秒数
◦ “usec” - 微秒数
◦ “minuteswest” - 格林威治向西的分钟数
◦ “dsttime” - 夏令时修正的类型

所以想打印单个的值可用 echo settimeofday[sec];

settimeofday()可以传入一个真值1，返回一个浮点数而不是数组

print_r(gettimeofday());
echo gettimeofday(1);

结果：

Array
(
    [sec] => 1479027864
    [usec] => 778749
    [minuteswest] => -60
    [dsttime] => 0
)
1479027864.7789

浮点数和上边是相对应的，1479027864说白了是时间戳

扩展： microtime() 返回当前 Unix 时间戳和微秒数，也就是gettimeofday()的前两个

作用：计算脚本运行时间（指的是很小很小的时间的差）

function fn(){
    list($a,$b) = explode(' ', microtime());    //以空格分割字符串，将分割后的字符串分别赋值给$a,$b
    return $a+$b;
}
$start = fn();
for ($i=0; $i<100000000 ; $i++) {     //时间越大，运行时间越大
    # code...
}
$end = fn();

echo round($end - $start,4);

运行此脚本的时间是3.1149秒

四：getdate()函数接受一个时间戳，返回一个人们能看得懂的时间

1. getdate()不接受时间戳参数时，返回当前时间的一个数组

print_r(getdate());

结果：

Array
(
    [seconds] => 39
    [minutes] => 10
    [hours] => 10
    [mday] => 13
    [wday] => 0
    [mon] => 11
    [year] => 2016
    [yday] => 317
    [weekday] => Sunday
    [month] => November
    [0] => 1479028239
)

2. getdate(1479027864)接受一个时间戳，返回的是时间戳指示的时间的详细信息数组

print_r(getdate(1479027864));

结果：

Array
(
    [seconds] => 24
    [minutes] => 4
    [hours] => 10
    [mday] => 13
    [wday] => 0
    [mon] => 11
    [year] => 2016
    [yday] => 317
    [weekday] => Sunday
    [month] => November
    [0] => 1479027864
