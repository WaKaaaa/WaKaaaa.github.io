php学习--日期与时间2

  view

五： time()返回当前的Unix时间戳，可以设置过去和未来的时间

echo time();

结果：

1479028890

time()函数的作用非常大

六：mktime()返回指定日期的时间戳

格式：mktime(时，分，秒，月，日，年);

$time = mktime(8,8,8,8,8,2008);
echo date('Y-m-d H:i:s',$time);

结果：2008-08-08 08:08：08

七：strtotime()将人们可读的日期转化为时间戳（和getdate()相反），用这个函数的频率高

用它可计算时间差

$time = strtotime('2016-11-13 20:20:20');
$now = time();
// echo time();

// 计算时间差，返回的是秒
echo $time-$now;

八：getlastmod()返回当前文件的最后修改时间的时间戳

echo getlastmod();    //返回时间戳
echo date('Y-m-d H:i:s',getlastmod());    //转化为可读的时间

扩展：获取文件最后修改时间的另一个函数是：filemtime()

九：localtime()取得本地时间数据，返回一个数组（此函数不常用，月份有问题）

date_default_timezone_set('Asia/shanghai');
print_r(localtime(time(),true));

结果：

Array
(
    [tm_sec] => 41
    [tm_min] => 10
    [tm_hour] => 20
    [tm_mday] => 13
    [tm_mon] => 10    //月份是从0开始的，也就是一月是0,11月是10（所以此函数不常用）
    [tm_year] => 116
    [tm_wday] => 0
    [tm_yday] => 317    //已经过去的天数
    [tm_isdst] => 0
)

十:设置时区

时区的概念：全球有24个时区。在国际中，使用一个统一的时间，叫通用协调时间（UTC），它与格林威治标准时间（GMT）相同，采用的是0时区

1.修改php.ini文件date.timezone=

echo date_default_timezone_get();    //获取当前时区
echo "<br>";
echo date('Y-m-d H:i:s');

echo "<br>";
date_default_timezone_set('Asia/shanghai');        //设置时区为上海时区
echo date('Y-m-d H:i:s');

结果：

Europe/Berlin
2016-11-13 13:01:25
2016-11-13 20:01:25
