java算法--棋盘覆盖

  view

问题描述：在2^k * 2^k的棋盘中有一个特殊的方格，用L型骨牌覆盖除了这个特殊方格之外的其他方格。要求骨牌不能重叠。

用0代表特殊方格，相同的数字代表同一个骨牌，不同骨牌用不同数字表示

算法思路：采用分治法，先将棋盘分成四个小棋盘，那么特殊方格肯定在这四个小棋盘中的一个里边。将没有特殊方格的三个棋盘交界地方覆盖上一个骨牌，使这个骨牌成为特殊方格。再将小棋盘切成4个小小棋盘，寻找没有特殊方格的三个棋盘覆盖，最终全部覆盖完毕。

具体的覆盖方法：

  1. 如果左上棋盘无特殊方格，将左上棋盘的右下角覆盖

  2. 如果左下棋盘无特殊方格，将左下棋盘的右上角覆盖

  3. 如果右上棋盘无特殊方格，将右上棋盘的左下角覆盖

  4. 如果右下棋盘无特殊方格，将右上棋盘的左下角覆盖

代码实现：

public class chessBoard {
// 设置棋盘规格为8
public static int size = 8;
// 设置特殊方格的横坐标（从0开始）为4
public static int dr = 4;
// 设置特殊方格的纵坐标（从0开始）为3
public static int dc = 3;
// 骨牌，从1开始，相同的骨牌用相同的数字，不同的骨牌数字不同，递增。骨牌个数为（8*8-1）/3=21个
public static int tile = 1;
public static int[][] board;


public static void chessBoard(int tr, int tc, int dr, int dc, int size) {
    // 分别是：左上行号，左上列号，特殊方格行号，特殊方格列号，棋盘规格2^k
    if (size == 1)    
        return;
    int t = tile++, // tile是全局变量，表示L型骨牌的编号
    s = size / 2; // 将棋盘分成上左，上右，下左，下右四部分
    // 判断特殊方格是否在左上角的小棋盘中
    if (dr < tr + s && dc < tc + s) {
        // 特殊方格在此盘中，则再次调用自身函数缩小范围
        chessBoard(tr, tc, dr, dc, s);
    } else {
        // 此盘中无特殊方格
        // 用t号L型骨牌覆盖右下角
        board[tr + s - 1][tc + s - 1] = t;    //这里是将t号骨牌覆盖到bord[3][3]的位置上（也就是右下角）
        chessBoard(tr, tc, tr + s - 1, tc + s - 1, s);    //这里特殊方格变成了3,3，将(0,0,3,3,4)这个棋盘分割
    }
    // 右上角（同理）
    if (dr < tr + s && dc >= tc + s) {    //特殊方格在此盘
        chessBoard(tr, tc + s, dr, dc, s);
    } else {
        board[tr + s - 1][tc + s] = t;
        chessBoard(tr, tc + s, tr + s - 1, tc + s, s);
    }
    // 左下角
    if (dr >= tr + s && dc < tc + s) {
        chessBoard(tr + s, tc, dr, dc, s);
    } else {
        board[tr + s][tc + s - 1] = t;
        chessBoard(tr + s, tc, tr + s, tc + s - 1, s);
    }
    // 右下角
    if (dr >= tr + s && dc >= tc + s) {
        chessBoard(tr + s, tc + s, dr, dc, s);
    } else {
        board[tr + s][tc + s] = t;
        chessBoard(tr + s, tc + s, tr + s, tc + s, s);
    }
}

public static void main(String[] args) {
    // 初始化棋盘
    board = new int[size][size];
    // 调用方法
    chessBoard(0, 0, dr, dc, size);
    // 显示棋盘
    for (int[] is : board) {
        System.out.println();
        for (int i : is) {
            System.out.printf("%4d", i);
         }
     }
 }

}

算法的时间复杂度：

设T(k)是算法chessBoard覆盖2^k * 2^k棋盘所需要的时间，从算法的分割策略可知，T(k)满足：

1. k=0, T(k) = O(1)

2. k>1, T(k) = 4T(k-1)+O(1)

解得T(k) = O(4^k)
