java算法学习--斐波那契数列

  view

斐波那契数列：1 1 2 3 5 8 11······

也就是从第二个开始，前两个数的和等于第三个数

public class Feonacci {
    //计算第n个菲波那切数
    public static int febonacci(int n)
    {
        if(n<=1) return 1;
        return febonacci(n-1)+febonacci(n-2);
    }

    public static void main(String[] args)
    {
        System.out.println("第五个斐波那契数是："+febonacci(5));
    }
}
