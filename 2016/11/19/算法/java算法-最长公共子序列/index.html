java算法学习--最长公共子序列

  view

1. 公共子序列：给定两个序列X和Y，如果另一个序列Z既是X的子序列，又是Y的子序列，那么就称Z是X，Y的公共子序列

2. 子序列：在给定的一个序列 X 删除掉若干个元素之后组成的序列 Z 是序列 X 的子序列。这个子序列是严格按照原来X序列下标递增的序列。简单的例子：Z = {B,C,D,B}是序列X = {A,B,C,B,D,A,B}的子序列，递增下标序列是{2,3,5,7}

3. 最长公共子序列：两个序列中最长的子序列。

4. 最长公共子序列有最优子结构性质
