PHP的基本知识学习总结

一：PHP概述

PHP：超文本预处理程序，通俗地讲是制作网页的脚本程序，是以.php为后缀名，写动态网站，与数据库交互

PHP运行环境：windows+apache+php+mysql
        linux+apache+php+mysql

PHP文件构成：HTML div CSS PHP JavaScript jQuery ajax等

PHP工作原理：请求->Apache->执行PHP代码->响应

二：PHP环境搭建

用appServ集成环境或者独立搭建

三：php基础知识

  01.基本语法

<?php 
    ......
 ?>

  02.输出的几种方式和比较

  •   echo: 函数，无返回值（返回void），作用是打印字符串
  •   print: 函数，返回整型，作用是打印字符串
  •   printf: 函数，返回字符串的长度，作用是打印字符串

  •   sprintf: 函数，返回字符串将字符串保存在内存中（不在浏览器下显示）
    ###

    <?php 

    echo "hello php1";
echo "</br>";
print("hello php2");
echo "</br>";
echo print("hello php3</br>"); // 这句话的作用是先将print里的字符串打印出来，再打印一个字符串，然后echo出print的类型，返回真，在浏览器中显示为1
echo "</br>";
printf("hello php4</br>");
echo printf("hello php5</br>"); //返回字符串的长度
echo "</br>";
sprintf("hello php6<br>"); // 字符串保存在内存中，不在浏览器界面显示
echo sprintf("hello php7"); // 返回字符串

    ?>
    运行结果：

    hello php2
    hello php3
    1
    hello php4
    hello php5
    15

      04.标识符

      标识符是变量的名称，关于标识符，PHP定义了一些简单的规则：

      标识符可以是任何长度，而且可以由任何字母、数字、下划线组成。

      标识符不能以数字开始。

      在PHP中，标识符是区分大小写的。

      一个变量名称可以与一个函数名称相同。

      05.变量的数据类型

    Integer(整数)、Float(浮点数，也叫Double,双精度)、String(字符串)、Boolean(布尔)、Array(数组)、Object(对象).

    在PHP中无论是double还是float，在浏览器中显示的都是“double”

      07.检测变量

    大部分的可变函数都是用来测试一个函数的类型的。PHP中有两个最常见的函数，分别是gettype()和settype()。
    这两个函数返回的string类型，也就是变量的类型字符串。
    用法：gettypt($变量)

    settype($变量，要转换的数据类型)

    isset()用来判断一个变量是否存在，返回的是布尔值true或false
    unset()用来销毁一个变量（如果变量存在则销毁，如果不存在此函数不发挥作用）

empty()用来判断一个变量的值是否为空，如果为空则为true否则为false。
换句话说，""、0、"0"、NULL、FALSE、array()、var $var; 以及没有任何属性的对象都将被认为是空的

PHP还提供了一些特定类型的测试函数。每一个函数都使用一个变量座位其参数，并且返回true或false。
is_array()、is_double()、is_float()、is_real()、is_long()、is_int()、is_integer()、is_string()、is_object()、is_resource()、is_null()、is_numeric() 

可以通过调用一个函数来实现转换变量数据类型的目的。
intval()、floatval()、strval();
注意：在浏览器中如果是真（true），返回1，如果是假（false），返回空（什么都不返回）

  08.php常量

常量一旦被定义之后，就不能再次更改。

语法：define（“常量名”，常量值）

###    define(“TOTAL”,199);

除了自定义常量外，PHP还预定了许多常量。了解这些常量的简单方法就是运行phpinfo()命令。

变量和常量的另一个差异在于常量只可以保存布尔值、整数、浮点数或字符串数据。这些类型都是标量数据。

  09.php中单双引号的区别

  双引号可以解析变量，即双引号中包含的变量会自动替换成实际的数值

  双引号可以解析转义字符，如\n等（这些字符在浏览器的显示中没有很大的区别，但是在页面元素即HTML中有变化）

  单引号按照普通字符串直接输出，他不会解析变量和转义字符，但是他速度快，用的多。无特殊说明推荐使用单引号
