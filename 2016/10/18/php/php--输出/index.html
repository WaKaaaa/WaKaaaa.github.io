php学习--echo print_r,printf,sprintf,print的比较

  view

php几种输出方式比较

  •   echo: 函数，无返回值（返回void），作用是打印字符串，还可以打印函数的类型和数组类型
  •   print: 函数，返回整型，作用是打印字符串
  •   printf: 函数，返回字符串的长度，作用是打印字符串
  •   sprintf: 函数，返回字符串将字符串保存在内存中（不在浏览器下显示）

  •   print_r: 函数，用来打印数组中的所有元素
    ###

    <?php 

    echo "hello php1";
echo "</br>";
print("hello php2");
echo "</br>";
echo print("hello php3</br>"); // 这句话的作用是先将print里的字符串打印出来，再打印一个字符串，然后echo出print的类型，返回真，在浏览器中显示为1
echo "</br>";
printf("hello php4</br>");
echo printf("hello php5</br>"); //返回字符串的长度
echo "</br>";
sprintf("hello php6<br>"); // 字符串保存在内存中，不在浏览器界面显示
echo sprintf("hello php7"); // 返回字符串
echo"<br>";

$userAges = array('张三' => 22, '李四' => 21, '王五'=>18);
echo $userAges; //返回$userAges的类型 array

    ?>
    运行结果：
    hello php1
    hello php2
    hello php3
    1
    hello php4
    hello php5
    15
    hello php7
    array
