遍历数组

遍历数组的方式：

  • for 循环遍历数组
  • foreach 遍历数组
  • list 遍历数组
  • each 遍历数组

    1.for循环遍历数组

    // for循环访问数组
    // 数组下标从0开始可用
    for ($i=0; $i < count($userName); $i++) { 
    # code...
echo $i."---".$userName[$i]."<br>";    //count方法取得数组的长度，sizeof是他的别名，一般用count
    }
    结果是： 0—张三
    1---李四
2---王五
3---赵六
4---嘻嘻嘻
5---嘿嘿嘿

2.foreach 遍历数组

//这种方法好处是在数组的下标不是从0开始或者根本不是数字的时候可用
foreach ($userName as $key => $value) { //此处的key和value可以替换成任意的变量名，只需要将下面输出统一即可，也可以单独打印下标或者值
    # code...
    echo $key."---".$value."<br>";
}

       0---张三
       1---李四
       2---王五
       3---赵六
       4---嘻嘻嘻
       5---嘿嘿嘿

3.each 遍历数组

<?php 
    $userAges = array('张三' => 22, '李四' => 21, '王五'=>18);
    // print_r($userAges);
    //each的使用
    //each --  返回数组中当前的键／值对并将数组指针向前移动一步 
    //这里有一个指针，默认情况下，指针是指向第一个键值对
    //这里的第一个键值对是 ,张三=>22
    //如果each($userAges)，那么获取的就是第一个键值对 '张三'=>22
    //each这个函数返回的是一个数组，
    //each将第一个键值对获取到，然后包装成一个新数组。

    each($userAges); //这是第一步，将张三，22取出来，包装成一个新的数组
    $a = each($userAges);    //这是第二步，将李四，21取出
    print_r($a)."<br>";
    echo $a[0];
    // 第一步的结果：Array ( [1] => 22 [value] => 22 [0] => 张三 [key] => 张三 ) 张三
    // 第二步的结果：Array ( [1] => 21 [value] => 21 [0] => 李四 [key] => 李四 ) 李四

 ?>

each循环打印

while (!!$a = each($userAges)) { //当是真，即括号里的值是1的时候执行以下语
    # code...
    echo $a['key']."---".$a['value']."<br>";
}

！！的作用：将真实存在的数值转换成布尔值，在这里each($userAges)返回的是array，即数组里有数据，有数据，用！！可以将这个array转化成1

4.list 遍历数组

<?php 
    $userAges = array("张三","李四","王五");
    // print_r($userAges);
    // 把一个数组中的值赋给一个变量
    list($a,$b) = $userAges;    //这里将数组userAges的第一个值赋给$a,第二个值赋给$b，其中$a,$b的值可以改编为任意的变量
    echo $a."<br>";
    echo $b."<br>";

    // 注意：list只能识别下标是0,1,2等数字的数组，自定义的key不能识别的
    // 如果一定要打印数组的键和值，就用each和list的组合
    $userAge = array('张三' => 22, '李四' => 21, '王五'=>18);
    list($c,$d) = each($userAge);    // each将张三，22取出来形成一个新的数组，0($c)表示key，也就是张三，1($d)表示value，也就是 22
    echo $c;    // 打印张三
    echo $d;    // 打印22
 ?>

5.array相关函数

5.1 reset()函数，将数组指针指向第一位

<?php 
    $userAges = array('张三' => 22, '李四' => 21, '王五'=>18);
    $a = each($userAges);    // 指向第一位,$a是第一位张三22的新数组 ，而$userAges现在指向的是第二位

    echo $a['key'];
    $b = each($userAges);
    echo $b['key'];

    reset($userAges);    // 将数组指针指向数组第一位，无论刚开始在第几位
    $c = each($userAges);
    echo $c['value'];
 ?>

5.2 array_unique函数，去除数组中的重复项

<?php 
    $numbers = array(1,2,5,3,6,2,4,7,4,2,5,2,5,1,3,6,7);
    print_r($numbers);

    $new_numners = array_unique($numbers);    //数组去重
    print_r($new_numners);
 ?>

5.3 array_flip()函数，置换数组的键和值

$userAges = array('张三' => 22, '李四' => 21, '王五'=>18);
$new_userAges = array_flip($userAges);
print_r ($new_userAges);
结果：Array
(
    [22] => 张三
    [21] => 李四
    [18] => 王五
)
