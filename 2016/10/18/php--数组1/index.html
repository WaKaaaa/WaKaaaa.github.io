php学习--数组1

PHP数组

1.数组命名规则

开头是$，后面跟字母或者下划线，其后的为字母，数字，下划线，不可以有其他元素

$1fu = array('1','2');    //不可以数字开头
$fu_&hello = array("hello","world");    //含有特殊符号&
$userName = array('张三','李四','王五','赵六','嘻嘻嘻','嘿嘿嘿'); //正确的命名

注意：在一个程序中，标量变量和数组变量不能同名，例如已经存在$userName的变量，再创建$userName的数组，后者会覆盖前者

数组变量的名称是区分大小写的，$user和$User是不同的两个变量数组类名

PHP常量和变量都区分大小写

<?php 
    $a = "hello";
    $A = "world";
    echo $a."<br>";
    echo $A."<br>";
    // 结果是 hello world，变量区分大小写，否则后者会覆盖前者

    define("c", 123);
    define("C", 456);
    echo c."<br>";
    echo C;
    // 都打印，常量也区分大小写

 ?>

PHP的类名和方法名甚至一些关键字都不区分大小写

2.创建数组的几种方法

// 创建数组的第一种方法：array函数创建
 $userName = array('张三','李四','王五','赵六','嘻嘻嘻','嘿嘿嘿');
 echo $userName."<br>";    // 用echo只能打印$userName的返回类型是array（数组）
 print_r($userName);    // 用print_r可以将数组打印出来
 echo "<br>";
 echo $userName[2]."<br>";    //在使用数据时，可以直接取他的下标，从0开始（0到n-1）

 // range自动创建一个数组，range(low,heigh)
 $number = range(0,10);    //自动创建0到10的数组
 print_r($number);
 echo "<br>";
 $letter = range(a, z);    //创建a到z的数组
 print_r($letter);
 echo "<br>";
 $reset = range(z, a);    //创建a到z的逆序数组
 print_r($reset);
 echo "<br>";

 // 利用数组标识符[]声明数组
 // 开发时不知数组的大小或者数组会改变，用此方法
// 用此种方法创建数组要求同一数组名相同，也可以添加数组
 $array[0] = "hello";
 $array[1] = "world";
 $array[2] = "and";
 $array[3] = "php";
 print_r($array);
