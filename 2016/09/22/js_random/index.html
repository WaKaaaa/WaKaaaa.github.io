JavaScript的random()函数

函数语法： Math.random()

函数参数：没有参数

函数返回值： [0,1)，即0到一之间的随机数，包括0但是不包括1

函数的用途： 生成0到1之间的随机数，主要用于随机验证码的生成

简单的小例子：

<script type = "text/javascript">
    alert(Math.random());
</script>

这是最简单的random()运用，弹出一个随机数的对话框

random()返回的是浮点数，但是在很多时候我们期待保留小数点的后n位小数，做法是：

1：toFixed()方法

<script type = "text/javascript">
var num = Math.random();
var newNum = num.toFixed(2);
alert(newNum);
// toFixed保留小数点后的位数,此处是保留小数点后两位
</script>

2：parseInt()方法

parseInt(num*100)/100

JS取整方法：

1. parseInt()函数丢弃小数部分

<script type = "text/javascript>
alert(parseInt(5/2));
<script/>

此函数的返回结果是2，可见parseInt()函数直接将小数部分截去

与之相对的是parseFloat()函数，可以将整数转换为浮点数

2.Math.ceil()函数向上取整

alert(Math.ceil(9/2));

此函数的返回值是5，即有小数的在小数部分+1

3.Math.floor()函数向下取整

alert(Math.floor(9/2));

此函数的返回值是4，与parseInt()类似

4.Math.round()函数四舍五入

alert(Math.floor(7/3));

此函数的返回值是2

1.猜数字游戏

思路：

  • 系统先随机产生一个数字
  • 用prompt()输入你猜的数字
  • 判断你输入的数字是否是数字类型，不是的话继续输入
  • 逻辑判断用if else 语句判断输入的数字大，小，还是正好
  • alert()说明结果直至游戏结束
<html>
<head>
    <title>猜数字游戏</title>
    <meta charset = "utf-8">
</head>
<body>
    <script type = "text/javascript">
    var RandomNum = parseInt(Math.random()*10);
    function guess()
    {
        var num = prompt("enter a number","");
        alert(RandomNum);
        var result = num - RandomNum;
        // alert(result);

        if (isNaN(num)) {
            alert("你输入的不是数字，请重新输入！");
            guess();
        }else if (result == 0) {
            alert("猜对了！");
        }else if (result>0) {
            alert("太大");
            guess();
        }else{
            alert("太小");
            guess();
        }

    }
    </script>

    <input type="button" value = "guess number" onclick = "guess();">
</body>
</html>

统计共几次猜中的代码：

<html>
<head>
    <title>猜数字游戏</title>
    <meta charset = 'utf-8'>
    <script type = "text/javascript">
        function play(){
            var time = 0;
            var result;
            var count = parseInt(Math.random()*10);
            alert(count);

        while(true){
            time++;
            var num = prompt("请输入0到10之间的整数","");
            if(isNaN(num)){
                alert("输入的不是数字！");
                //不是数字，太大，太小都会继续循环，直到得出正确结果才跳出循环
            }else{
                result =count - num;

                if (result == 0) {
                    alert("猜对了，共"+time+"次");
                    break;
                }else if (result>0) {
                    alert("太小！");
                }else{
                    alert("太大！");
                }
        }
    }
}
    </script>
</head>

<body>
    <input type="button" value = "猜数字" onClick = "play();">
</body>

解析：因为Matn.random()函数只能产生[0,1)之间的随机数，*10可以保证产生的数字在[0,10)之间。

isNaN(num)来判断输入的类型是否是数字，如果是数字则返回false，如果不是数字类型则返回true

2.模仿微信红包游戏（输入总金额和份数，输出每一份的随机金额，最小金额为0.01）

思路：

  • prompt()输入红包金额和红包个数
  • 由于random()函数正好可以产生0到1的随机数，那么可以用来计算百分比，即总金额*random可得第一次红包的额度
  • 由于红包是一个一个有先后顺序地被抢，所以可以定义一个left变量来存储每次剩下的红包金额
  • 定义一个数组存储每次的红包金额
  • parseInt()每个红包精确到小数点后两位
<html>
<head>
    <title>随机红包</title>
    <meta charset = 'utf-8'>
    <script type = "text/javascript">
        var jine = prompt("请输入红包金额","");
        var geshu = prompt("请输入红包个数","");
        var left = jine;
        var arr = new Array();

        for(i = 0; i<geshu-1;i++){
            // 保证金额随机的方法是：用户金额范围在0.01~剩余平均数*2之间
            arr[i] = parseInt((left*2/(geshu-i))*Math.random()*100)/100;
            left -= arr[i];
            document.write(arr[i]+"<br/>");
        }
        document.write(parseInt(left*100)/100);

    </script>
</head>
</html>

解析：为使每个人的红包金额差别不是太大，每个红包的金额范围是[0.01~剩余红包金额/剩余个数*2]

3.随机产生0~99之间的整数，并按降序进行排列

<html>
<head>
    <title>随机产生数字并升序排列</title>
    <meta charset = "utf-8">
</head>
<body>
      <script type = "text/javascript">
           var arr = new Array();
           for(i=0;i<5;i++){
               arr[i] = parseInt(Math.random()*100);
           }
           document.write("排序前："+arr+"</br>");
           document.write("排序后："+arr.sort(sortNum));

           function sortNum (a,b){
               if (a>b) {
                   return -1;
               }else if(a<b){
                   return 1;
               }else{
                   return 0;
               }
           }
    </script>
</body>
</html>

解析：sort是对数组进行升序排序，属于Array对象的一个方法，定义function sortNum函数的原因是sort()方法如果没有指定参数，对字符编码的顺序对数组中的元素进行排序。 定义函数之后可以转为对数字的排序。

  • 若a小于b，返回1，在排序后的数组中a应该在b的后边
  • a等于b，返回0，a,b的位置不变
  • a大于b，返回-1，a在b的前边
  • 由此实现降序排列，升序排列只是返回-1和1的条件相互交换即可

    简便方法：

    function sortNum (a,b){
              return b-a;
}

    注意：return b-a 不是返回一个相减的值，JS规定，return b-a 是比较a,b的值
