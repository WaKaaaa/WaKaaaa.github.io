      07.检测变量

    大部分的可变函数都是用来测试一个函数的类型的。PHP中有两个最常见的函数，分别是gettype()和settype()。
    这两个函数返回的string类型，也就是变量的类型字符串。

    settype($变量，要转换的数据类型)

    isset()用来判断一个变量是否存在，返回的是布尔值true或false
    unset()用来销毁一个变量（如果变量存在则销毁，如果不存在此函数不发挥作用）

empty()用来判断一个变量的值是否为空，如果为空则为true否则为false。

PHP还提供了一些特定类型的测试函数。每一个函数都使用一个变量座位其参数，并且返回true或false。
is_array()、is_double()、is_float()、is_real()、is_long()、is_int()、is_integer()、is_string()、is_object()、is_resource()、is_null()、is_numeric() 

可以通过调用一个函数来实现转换变量数据类型的目的。
intval()、floatval()、strval();

  08.php常量

常量一旦被定义之后，就不能再次更改。

语法：define（“常量名”，常量值）

###    define(“TOTAL”,199);

除了自定义常量外，PHP还预定了许多常量。了解这些常量的简单方法就是运行phpinfo()命令。

变量和常量的另一个差异在于常量只可以保存布尔值、整数、浮点数或字符串数据。这些类型都是标量数据。

  09.php中单双引号的区别

  双引号可以解析变量，即双引号中包含的变量会自动替换成实际的数值

  双引号可以解析转义字符，如\n等（这些字符在浏览器的显示中没有很大的区别，但是在页面元素即HTML中有变化）

  单引号按照普通字符串直接输出，他不会解析变量和转义字符，但是他速度快，用的多。无特殊说明推荐使用单引号

    2.创建数组：

  • new Array();不带参数，返回空数组，length = 0。如： var arr = new Array();
  • Array(x); x的取值可为任何数，返回大小是x的数组，length = x。如： var arr = new Array(5);表示此数组的长度是5
  • new Array[x1,x2,x3...];x1,x2,x3..指的是数组的元素，返回长度为参数个数的数组，length = 个数，如： var arr = new Array[1,'2',3,4,5,6.8,7]; 表示此数组的长度是6，数组中的参数类型可以不相同

  • concat()方法：连接两个或者更多的数组，并返回合并后的新数组
    用法：arrayObj.context(array x,array x….);

    一个数组和其他元素相连

    var arr1 = [1,2,3];
var arr2 = [4,5,6];

JavaScript学习--random()函数

JavaScript的random()函数

函数语法： Math.random()

函数参数：没有参数

函数返回值： [0,1)，即0到一之间的随机数，包括0但是不包括1

函数的用途： 生成0到1之间的随机数，主要用于随机验证码的生成

简单的小例子：

<script type = "text/javascript">
    alert(Math.random());
</script>

这是最简单的random()运用，弹出一个随机数的对话框

random()返回的是浮点数，但是在很多时候我们期待保留小数点的后n位小数，做法是：

1：toFixed()方法

<script type = "text/javascript">
var num = Math.random();
var newNum = num.toFixed(2);
alert(newNum);
// toFixed保留小数点后的位数,此处是保留小数点后两位
</script>

2：parseInt()方法

parseInt(num*100)/100

JS取整方法：

1. parseInt()函数丢弃小数部分

<script type = "text/javascript>
alert(parseInt(5/2));
<script/>

此函数的返回结果是2，可见parseInt()函数直接将小数部分截去

与之相对的是parseFloat()函数，可以将整数转换为浮点数

2.Math.ceil()函数向上取整

alert(Math.ceil(9/2));

此函数的返回值是5，即有小数的在小数部分+1

3.Math.floor()函数向下取整

alert(Math.floor(9/2));

此函数的返回值是4，与parseInt()类似

4.Math.round()函数四舍五入

2.模仿微信红包游戏（输入总金额和份数，输出每一份的随机金额，最小金额为0.01）

思路：

  • prompt()输入红包金额和红包个数
  • 由于random()函数正好可以产生0到1的随机数，那么可以用来计算百分比，即总金额*random可得第一次红包的额度
  • 由于红包是一个一个有先后顺序地被抢，所以可以定义一个left变量来存储每次剩下的红包金额
  • 定义一个数组存储每次的红包金额
  • parseInt()每个红包精确到小数点后两位
<html>
<head>
    <title>随机红包</title>
    <meta charset = 'utf-8'>
    <script type = "text/javascript">
        var jine = prompt("请输入红包金额","");
        var geshu = prompt("请输入红包个数","");
        var left = jine;
        var arr = new Array();

        for(i = 0; i<geshu-1;i++){
            // 保证金额随机的方法是：用户金额范围在0.01~剩余平均数*2之间
            arr[i] = parseInt((left*2/(geshu-i))*Math.random()*100)/100;
            left -= arr[i];
            document.write(arr[i]+"<br/>");
        }
        document.write(parseInt(left*100)/100);

    </script>
</head>
</html>

解析：为使每个人的红包金额差别不是太大，每个红包的金额范围是[0.01~剩余红包金额/剩余个数*2]

解析：sort是对数组进行升序排序，属于Array对象的一个方法，定义function sortNum函数的原因是sort()方法如果没有指定参数，对字符编码的顺序对数组中的元素进行排序。 定义函数之后可以转为对数字的排序。

  • 若a小于b，返回1，在排序后的数组中a应该在b的后边
  • a等于b，返回0，a,b的位置不变
  • a大于b，返回-1，a在b的前边
  • 由此实现降序排列，升序排列只是返回-1和1的条件相互交换即可

    简便方法：

    function sortNum (a,b){
              return b-a;
}

    注意：return b-a 不是返回一个相减的值，JS规定，return b-a 是比较a,b的值
