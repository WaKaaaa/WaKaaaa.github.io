java算法--棋盘覆盖

问题描述：在2^k * 2^k的棋盘中有一个特殊的方格，用L型骨牌覆盖除了这个特殊方格之外的其他方格。要求骨牌不能重叠。

用0代表特殊方格，相同的数字代表同一个骨牌，不同骨牌用不同数字表示

算法思路：采用分治法，先将棋盘分成四个小棋盘，那么特殊方格肯定在这四个小棋盘中的一个里边。将没有特殊方格的三个棋盘交界地方覆盖上一个骨牌，使这个骨牌成为特殊方格。再将小棋盘切成4个小小棋盘，寻找没有特殊方格的三个棋盘覆盖，最终全部覆盖完毕。

具体的覆盖方法：

  1. 如果左上棋盘无特殊方格，将左上棋盘的右下角覆盖

  2. 如果左下棋盘无特殊方格，将左下棋盘的右上角覆盖

  3. 如果右上棋盘无特殊方格，将右上棋盘的左下角覆盖

  4. 如果右下棋盘无特殊方格，将右上棋盘的左下角覆盖

代码实现：

public class chessBoard {
// 设置棋盘规格为8
public static int size = 8;
// 设置特殊方格的横坐标（从0开始）为4
public static int dr = 4;
// 设置特殊方格的纵坐标（从0开始）为3
public static int dc = 3;
// 骨牌，从1开始，相同的骨牌用相同的数字，不同的骨牌数字不同，递增。骨牌个数为（8*8-1）/3=21个
public static int tile = 1;
public static int[][] board;


public static void chessBoard(int tr, int tc, int dr, int dc, int size) {
    // 分别是：左上行号，左上列号，特殊方格行号，特殊方格列号，棋盘规格2^k
    if (size == 1)    
        return;
    int t = tile++, // tile是全局变量，表示L型骨牌的编号
    s = size / 2; // 将棋盘分成上左，上右，下左，下右四部分
    // 判断特殊方格是否在左上角的小棋盘中
    if (dr < tr + s && dc < tc + s) {
        // 特殊方格在此盘中，则再次调用自身函数缩小范围
        chessBoard(tr, tc, dr, dc, s);
    } else {
        // 此盘中无特殊方格
        // 用t号L型骨牌覆盖右下角
        board[tr + s - 1][tc + s - 1] = t;    //这里是将t号骨牌覆盖到bord[3][3]的位置上（也就是右下角）
        chessBoard(tr, tc, tr + s - 1, tc + s - 1, s);    //这里特殊方格变成了3,3，将(0,0,3,3,4)这个棋盘分割
    }
    // 右上角（同理）
    if (dr < tr + s && dc >= tc + s) {    //特殊方格在此盘
        chessBoard(tr, tc + s, dr, dc, s);
    } else {
        board[tr + s - 1][tc + s] = t;
        chessBoard(tr, tc + s, tr + s - 1, tc + s, s);
    }
    // 左下角
    if (dr >= tr + s && dc < tc + s) {
        chessBoard(tr + s, tc, dr, dc, s);
    } else {
        board[tr + s][tc + s - 1] = t;
        chessBoard(tr + s, tc, tr + s, tc + s - 1, s);
    }
    // 右下角
    if (dr >= tr + s && dc >= tc + s) {
        chessBoard(tr + s, tc + s, dr, dc, s);
    } else {
        board[tr + s][tc + s] = t;
        chessBoard(tr + s, tc + s, tr + s, tc + s, s);
    }
}

public static void main(String[] args) {
    // 初始化棋盘
    board = new int[size][size];
    // 调用方法
    chessBoard(0, 0, dr, dc, size);
    // 显示棋盘
    for (int[] is : board) {
        System.out.println();
        for (int i : is) {
            System.out.printf("%4d", i);
         }
     }
 }

}

结果：

 3   3   4   4   8   8   9   9
 3   2   2   4   8   7   7   9
 5   2   6   6  10  10   7  11
 5   5   6   1   1  10  11  11
13  13  14   0   1  18  19  19
13  12  14  14  18  18  17  19
15  12  12  16  20  17  17  21
15  15  16  16  20  20  21  21

算法的时间复杂度：

设T(k)是算法chessBoard覆盖2^k * 2^k棋盘所需要的时间，从算法的分割策略可知，T(k)满足：

1. k=0, T(k) = O(1)

2. k>1, T(k) = 4T(k-1)+O(1)

解得T(k) = O(4^k)

java算法学习--排列问题 

public class MyMath {
    public static void perm(String[] list , int k,int m)
    {
        //k是第一个数字，m是最后一个数字
        //输出list[k...m]的所有排列方式
        if(k == m) {    
            //只有一个元素.一到递归的出口就输出数组，此数组为全排列
            for(int j=0; j<= m ;j++) {
                System.out.print(list[j]);
            }
            System.out.println();
        }
        else
            //还有很多元素，递归产生排列
        for(int i = k; i <= m ; i++) {
            //for循环将k~m中每个数放到k位置中去，
            MyMath.swap(list, k, i);
            //假设k位置确定，那么对k+1到m的数继续递归
            perm(list, k+1, m); 
            //换过去之后再还原
            MyMath.swap(list, k, i);
        }    
    }

    private static void swap(String[] str, int i, int first) {
        //temp变量实现元素的交换。
         String temp;
         temp = str[first];
         str[first] = str[i];
         str[i] = temp;      
    }

    public static void main(String args[]){ 
        //java中main()方法是Java应用程序的入口方法，JVM通过main方法找到需要启动的运行程序，
        //输入数组
         String[] str = {"a","b","c","d","e","f"};
         //从第一个到最后一个全排列，变换数字可以任意截取其中一部分，比如perm(str,0,2)，
         //只会对前三个数字进行全排列，其余类似
         perm(str, 0, 5);
         //输出list[0..2]的所有排列方式

} 

}

php学习--简单的随机图像验证码

完整代码：

ini_set("error_reporting", "E_ALL & ~E_NOTICE");
for ($i=0;$i<4;$i++) {
    $nmsg .= dechex(mt_rand(0,15));
}

header('Content-Type: image/png');
$im = imagecreatetruecolor(75,25);
$blue = imagecolorallocate($im,0,102,255);
$white = imagecolorallocate($im,255,255,255);
imagefill($im,0,0,$blue);
imagestring($im,5,20,5,$nmsg,$white);    //将四位验证码加入到图片中去
imagepng($im);
imagedestroy($im);

解析：

思路：首先生成四位随机的数字字母，这里可以用十六进制数字表示，然后将这四位随机数放在我们已经写好的图片里即可

生成四位数字

for ($i=0;$i<4;$i++) {
    $nmsg .= dechex(mt_rand(0,15));
}
// mt_rand(范围可选) 生成一个更好的随机数
// dechex()将十进制的随机数转化成十六进制

在这里会产生一个报错：Notice: Undefined variable: nmsg in D:\xampp1\htdocs\PHP\Basic14\demo2.php on line 4，修改的方法：

2. 在PHP头部加入代码：ini_set("error_reporting", "E_ALL & ~E_NOTICE");，也可以解决这类问题

php学习--处理图像

1. 为什么用PHP处理图像：

传统的web开发在页面中加入图片一般用HTML 的标记来实现，在PHP中为了提高页面的传输速率，便于对数据统计，有时采用GD2库或者jpgraph类库来绘制

2. 什么是GB2库

gd2函数库是一个公开源码的用于创建图形图像的函数库，该函数库由C语言编写，可以在Perl，PHP等多种语言中调试运用。PHP使用GD2库可以制作统计图，为图片添加水印及动态图表等

3. GD2创建画布的步骤

开发人员可以在已有的图像上绘制，也可以在没有任何素材的基础上绘制，在这种没有素材的情况下首先要创建画布，之后的所有操作都将依照所创建的画布运行

第一步：设置文件的MIME类型

header('Content-Type: image/png;');

一般生成的图像类型有 png jpg(jpeg) gif bmg

这句话的意思是输出网页的内容是png格式的图片

注:1：在网页制作中，默认的输出格式是：text/html，所以一般的网页不需要写输出格式

注意2：header之前不能有任何形式的输出，空格也不行

第二步：imagecreatetruecolor()新建一个新真彩色图像，返回一个图像标识，大小为x*y的黑色图像

$im = imagecreatetruecolor(200, 200);    // 传入两个参数：宽和高，单位是像素

第三步：imagecolorallocate(建好的图像，红，绿，蓝)为一个图像分配颜色 （其中红绿蓝的颜色取值为0到255）

$blue = imagecolorallocate($im,0,102,255);    //这里创建一个蓝色背景

imagefill(图像，横坐标，纵坐标，颜色) ：区域填充

imagefill($im,0,0,$blue);

第四步：在蓝色的背景上输入一些线条，文字等

$white = imagecolorallocate($im,255,255,255);
//imageline -- 画一条线段
//imageline(图像，起始横坐标，起始纵坐标，终止横坐标，终止纵坐标，颜色)
imageline($im,0,0,200,200,$white);
imageline($im,200,0,0,200,$white);
//imagestring -- 水平地画一行字符串
imagestring($im,5,80,20,'Hello World',$white);

第五步：输出最终图形

imagepng($im);
//这一步是必须的，很重要的，不写的话会出现错误
// 以 PNG 格式将图像输出到浏览器或文件

第六步：将所有的资源全部清空

imagedestroy($im);

php学习--操作中文字符串

总结：

必须使用mb_的是mb_strlen(),mb_strpos(),mb_substr()

strlen()求字符串的长度

mb_strlen()求中文字符串的长度，一个汉字代表一个字符

这里：gbk和Utf8都说变长编码

对于utf8，一个汉字可能是2-3个字节，不一定就是3个字节

一般在gbk下是两个字节，在Utf8下是3个字节

$str1 = '我叫jewel';
echo strlen($str1).'<br>';    //11，一个汉字代表三个字符
echo mb_strlen($str1,'utf8');    //7
// utf8可以省略，那么为内部编码方式

strstr()求某字符串到结尾的字符，返回在这个字符首次出现的位置开始到结尾组成的字符串

这个函数区分大小写，用stristr()不区分大小写

这个中文英文影响不大

$str2 = 'he is a good boy';
echo strstr($str2, 'i');    //is a good boy
$str3 = '我爱我的家乡';
echo mb_strstr($str3, '的');
echo strstr($str3, '我');

相对应的是strpos()，查找字符首次出现的位置

$str4 = '我爱我的家乡';
echo strpos($str4, '的');    //这种情况只能出现在3的倍数的字符串，比如‘爱’在第三位，‘的’在地9位，这时候就需要用mb_strpos()了
echo mb_strpos($str4,'爱');    //1
echo mb_strpos($str4,'的');    //3 这时候将一个汉字转化为一个字符了

substr(字符串，开始位置，子串长度（可选）)

$str5 = '我爱我的家乡';
echo substr($str5, 1,2);    //乱码，无法取
echo mb_substr($str5, 1,2);    //爱我

php学习--字符串比较

一：总结

1. strcmp()比较字符串的大小，区分大小写，对ascall码的比较

2. strcasecmp()比较字符串大小，不区分大小写，对ascall码的比较

3. strnatcmp()按照数字的顺序进行排序

4. strnatcasecmp()不区分大小写的自然排序

5. strspn()计算字符串中全部字符都存在于指定字符集合中的第一段子串的长度

6. strlen()函数测试字符串的长度

7. substr_count(字符串，待检验的字符串)返回一个字符串在另一个字符串中出现的次数。

8. strstr()用于在一个较长的字符串专供查找匹配的字符串或字符

9. stristr()是不区分大小写的

10. strpos()返回字符从大的字符串的第几个位置出现，从0开始

11. strrpos()返回第一次出现字符的倒数第几个的位置

详解：

1.strcmp()是对字符ASCALL码的比较，区分大小写

第一个字符小于第二个字符返回 -1

第一个字符等于第二个字符返回 0

第一个字符大于第二个字符返回 1

echo strcmp('a', 'b');    //返回-1

2. strcasecmp()不区分大小写的比较，对ASCALL码比较

echo strcasecmp('a', 'A');    //不区分大小写，返回0
echo strcmp('a', 'A');    //区分大小写，返回1

A的ascall码是65，a是97

3. strnatcmp()按照数字的顺序进行排序

默认情况下是对ASCALL的排序，比如2和10比较时，2大于10

echo strnatcmp('2', '10');    //按照数字排序，返回-1
echo strcmp('2', '10');        //这个是按照ascall码排序，返回1

4. strnatcasecmp()是不区分大小写的自然排序

5. strapn(待检查的字符串，检查字符列表，【待检查字符串开始的位置】，【待检查字符串检查的长度】)

计算字符串中全部字符都存在于指定字符集合中的第一段子串的长度

待检查的字符串：小字符串

检查字符串列表：大字符串，从这个里边搜索小串是否存在

待检查字符串开始的位置， 待检查字符串检查的长度， 可选可不选

echo strspn('boy','he is a good boy boy');    //返回3
echo strspn('d','he is a good boy boy');    //返回1，第一段子串中有一个d
echo strspn('yc60','yc60.com@gmail.com',1,2);    //从c6中检查出有两个字符符合，从第二个字符开始，查找两个长度的字符串，看看这一段有几个符合的字符

6. strlen()函数测试字符串的长度

echo strlen('hello world');    //11

7. substr_count(字符串，待检验的字符串)返回一个字符串在另一个字符串中出现的次数。

echo substr_count('hello world', 'o');    //o在字符串中出现两次
echo substr_count('o','hello world');    //hello world在o中出现0次

查找和替换字符串

8. strstr()用于在一个较长的字符串专供查找匹配的字符串或字符，从第一次找到这个字符开始算起，返回的是从待检验的字符往后的子字符串，strchr()是他的别名

echo strstr('hello world', 'o');    //o world
echo strchr('hello world', 'o');    //o world

9. stristr()是不区分大小写的

echo strstr('hellO world', 'o');    //O world

10. strpos()返回字符从大的字符串的第几个位置出现，从0开始

echo strpos('hello world', 'o');    //从第五个字符开始出现的，返回4

11. strrpos()返回第一次出现字符的倒数第几个的位置

echo strrpos('hello world', 'o');    //第一次出现o的时候位置的倒序，返回7

php学习--操作子字符串

总结：

1. explode()以某个字符分割字符串，返回字串组成的数组

2. join()将数组中的元素合并成字符串

3. strtok()标记分割字符串

4. substr()函数返回字符串的子串，不适合汉字

5. str_aplit()函数将字符串转化为数组，分解成一小段一小段的子字符串，不适合汉字

6. strrev()函数将字符串逆反过来，不适合汉字

详解：

1. explode(分割的符号，被分割的字符串) 以某个字符分割字符串，返回字串组成的数组

$str = '123456789@qq.com';
print_r(explode('@', $str));

2. join(合并的符号，被合并的数组) 将数组中的元素合并成字符串

$str2 = array('hello' ,'world','!');
echo join(' ',$str2);

3. implode()是join的别名，他俩是一样的作用

4. strtok()标记分割字符串，这个函数的好处是可以使用多个参数来分割字符串

// 注意：符号的顺序是无所谓的
// 仅第一次调用strtok的时候使用string参数，后来每次调用 strtok都只使用分隔符号参数（这里指的是,#%），因为他会记住他在字符串中的位置。
// 如果重新分割新的字符串，就需要再次调用string和strtok函数，来完成初始化工作
$str3 = 'hello, he is %a good # boy';
$tok = strtok($str3, ',#');
while ($tok) {
    echo $tok.'<br>';
    $tok = strtok('#,%');
}

5. substr(字符串，子字符串开始的下标，子字符串的长度) 返回字符串的子串

// 第一个参数是字符串，第二个参数是子字符串开始的下标（从0开始），第三个参数是子字符串的长度（可选），默认是到最后一个字符
// 注意这个函数不可以分割中文，因为中文是两个字节，这个函数分割的是一个字节
$str4 = 'hello he is a good boy';
echo substr($str4, 2, 5);

6. str_aplit(字符串，子字符串的长度) 将字符串转化为数组，分解成一小段一小段的子字符串

// 第一个参数是要分解的字符串，第二个参数是每一个字符串的长度（可选），默认是分割成单个的字符数组
$str5 = 'hello he is a good boy';
print_r(str_split($str5,2));

7. strrev()函数将字符串逆反过来

$str6 = 'hello he is a good boy';
echo strrev($str6);

php学习--字符串基本操作

总结：

1. trim()清除字符串左右两边的空格

2. ltrim()清除左边的空格

3. rtrim()清除右边的空格

4. nl2br()在字符串所有新行之前插入HTML换行标记（
）

5. htmlspecialchars()将特殊的字符转化成HTML实体

6. addslaashes()使用反斜杠引用字符串

7. strtoupper()函数将字符串转换为大写

8. strtolower()函数将字符串转换成小写

9. ucfirst()函数将第一个字母转换为大写

10. ucwords()函数将每个单词第一个字母转换为大写

11. str_pad()使用另一个字符串填充字符串为指定长度

一：trim()，ltrim()，rtrim()

//  trim()清除字符串左右两边的空格
$str = '       jewel       ';
echo trim($str).'<br>';
// ltrim()清除左边的空格
echo ltrim($str).'<br>';
// rtrim()清除右边的空格
echo rtrim($str);

二：nl2br()在字符串所有新行之前插入HTML换行标记（
）

注意转义字符只能包裹在双引号里边，这一点非常重要

转义字符包括：\t \n \r

$str1 = "I am Jewel\n hello world";
echo nl2br($str1);
// 打印出：I am Jewel<br/> hello world

echo nl2br("foo isn't\n bar");

$string = "This\r\nis\n\ra\nstring\r";
echo nl2br($string);

三：htmlspecialchars()将特殊的字符转化成HTML实体

$str2 = '<strong>jewel</strong>';
echo htmlspecialchars($str2);
// 
// strip_tags从字符串中去掉HTML和PHP标记
echo strip_tags($str2);

四：addslaashes()使用反斜杠引用字符串，为了数据可以查询语句等的需要在某些字符（’ “ \ null等）前加上了反斜杠。一般用在你想要向数据库中输入数据的时候

对于字符串来说，某些字符肯定是有效的，但是当将数据插入到数据库中的时候可能会引起一些问题，因为数据库会将这些字符解释成控制符。这些有问题的字符就是引号（单引号和双引）、反斜杠（\）和NULL字符。

PHP提供了两个专门用于转义字符串的函数。在将任何字符串写到数据库之前，应该使用addslashes()将它们重新格式化

$str3 = 'I am a good girl. "this" hello';
echo addslashes($str3);

五:

strtoupper()函数将字符串转换为大写

strtolower()函数将字符串转换成小写

ucfirst()函数将第一个字母转换为大写

ucwords()函数将每个单词第一个字母转换为大写

echo strtoupper('hello wworld').'<br>';
echo strtolower('HELLO world');
echo ucfirst('hello world');
echo ucwords('hello world');

六: str_pad()使用另一个字符串填充字符串为指定长度

// 第二个参数表示总共的位数
echo str_pad("hello", 10).'<br>';

$input = "Alien";
echo str_pad($input, 10).'<br>';                      // 输出 "Alien     "
echo str_pad($input, 10, "-=", STR_PAD_LEFT);  // 输出 "-=-=-Alien"
echo str_pad($input, 10, "_", STR_PAD_BOTH);   // 输出 "__Alien___"
echo str_pad($input, 6 , "___");               // 输出 "Alien_"
