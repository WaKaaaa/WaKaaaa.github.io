php学习--字符串基本操作

总结：

1. trim()清除字符串左右两边的空格

2. ltrim()清除左边的空格

3. rtrim()清除右边的空格

4. nl2br()在字符串所有新行之前插入HTML换行标记（
）

5. htmlspecialchars()将特殊的字符转化成HTML实体

6. addslaashes()使用反斜杠引用字符串

7. strtoupper()函数将字符串转换为大写

8. strtolower()函数将字符串转换成小写

9. ucfirst()函数将第一个字母转换为大写

10. ucwords()函数将每个单词第一个字母转换为大写

11. str_pad()使用另一个字符串填充字符串为指定长度

一：trim()，ltrim()，rtrim()

//  trim()清除字符串左右两边的空格
$str = '       jewel       ';
echo trim($str).'<br>';
// ltrim()清除左边的空格
echo ltrim($str).'<br>';
// rtrim()清除右边的空格
echo rtrim($str);

二：nl2br()在字符串所有新行之前插入HTML换行标记（
）

注意转义字符只能包裹在双引号里边，这一点非常重要

转义字符包括：\t \n \r

$str1 = "I am Jewel\n hello world";
echo nl2br($str1);
// 打印出：I am Jewel<br/> hello world

echo nl2br("foo isn't\n bar");

$string = "This\r\nis\n\ra\nstring\r";
echo nl2br($string);

三：htmlspecialchars()将特殊的字符转化成HTML实体

$str2 = '<strong>jewel</strong>';
echo htmlspecialchars($str2);
// 
// strip_tags从字符串中去掉HTML和PHP标记
echo strip_tags($str2);

四：addslaashes()使用反斜杠引用字符串，为了数据可以查询语句等的需要在某些字符（’ “ \ null等）前加上了反斜杠。一般用在你想要向数据库中输入数据的时候

对于字符串来说，某些字符肯定是有效的，但是当将数据插入到数据库中的时候可能会引起一些问题，因为数据库会将这些字符解释成控制符。这些有问题的字符就是引号（单引号和双引）、反斜杠（\）和NULL字符。

PHP提供了两个专门用于转义字符串的函数。在将任何字符串写到数据库之前，应该使用addslashes()将它们重新格式化

$str3 = 'I am a good girl. "this" hello';
echo addslashes($str3);

五:

strtoupper()函数将字符串转换为大写

strtolower()函数将字符串转换成小写

ucfirst()函数将第一个字母转换为大写

ucwords()函数将每个单词第一个字母转换为大写

echo strtoupper('hello wworld').'<br>';
echo strtolower('HELLO world');
echo ucfirst('hello world');
echo ucwords('hello world');

六: str_pad()使用另一个字符串填充字符串为指定长度

// 第二个参数表示总共的位数
echo str_pad("hello", 10).'<br>';

$input = "Alien";
echo str_pad($input, 10).'<br>';                      // 输出 "Alien     "
echo str_pad($input, 10, "-=", STR_PAD_LEFT);  // 输出 "-=-=-Alien"
echo str_pad($input, 10, "_", STR_PAD_BOTH);   // 输出 "__Alien___"
echo str_pad($input, 6 , "___");               // 输出 "Alien_"

php学习--session

一：会话

会话是指用户访问web站点上的时期内存储的用户操作的一些动作。PHP提供了管理会话的机制，可以追踪到用户在这一时期的动作，显示用户喜欢什么，需要什么。

会话与cookie类似，但是功能更加强大，可以管理大量数据，而且可以将信息保存在服务器端，安全，存储长度也没有限制

二：session的概念

session生命周期：从用户访问页面开始到断开与网络连接为止，用户每一次连接，PHP都会自动生成一个唯一的sessionID来标识当前用户，从而与其他用户进行区分。session会话时，sessionID分别保存在客户端和服务器端两个位置。

在客户端，使用临时的cookie保存在浏览器的指定目录中，称为session cookie

在服务器端，以文本文件的形式保存在指定的session目录中

网页是无状态的连接程序，无法记录用户的状态信息，所以用session记录用户的有关信息。

三：启动session

1.session_start()函数启动session，在使用session_start()之前浏览器不能有任何输出。

2.session_register()函数为会话登录一个变量来隐含的启动会话。设置php.ini文件里的register_globals 设为On

四：使用session

session有一个功能是保存当前用户的特定数据和相关信息，包括字符串，数组，对象等。使用超级全局变量$_SESSION[]将各种类型的数据添加到session中。

session_start();
$_SESSION['name'] = 'jewel';    //创建一个session

五：删除session

1.unset()删除单个会话

unset($_SESSION['name']);

注意：在使用unset()时，$_SESSION[‘name’]里的元素（这个里边是name）不能省略，也就是不能一次注销整个数组，这样会禁止整个会话的功能。unset($_SESSION)会将超级全局变量$_SESSION销毁，而且不能恢复，用户不可以再次注册$_SESSION变量

2.将空值赋给$_SESSION来删除所有会话

$_SESSION = array();

3.session_destory()结束当前会话

session_destory()函数清除结束当前的会话，并清除会话中的所有资源，彻底销毁session

php学习--cookie

一：cookie的概念

cookie是在http协议下（http是一种无状态的协议，每次请求的处理都与之前或者之后的请求无关），将服务器传递给浏览器少量的数据保存到用户浏览器的一种方式。通过这种方式，即使在浏览器被关闭和连接中断的情况下，用户仍然可以维护状态数据

通俗的将，cookie是保存在你的客户机上的一个小文件，它是经过加密的文件，保存着你的登录信息等

二：cookie作用域

1.记录访客的某些信息，如登录的用户名，访问网站的次数等

2.在页面之间传递变量。浏览器不会保存当前页面的任何变量信息，当浏览器被关闭之后，页面上的变量信息就会随之消失。这时候就可以把变量以cookie的形式保存下来，然后在下一页通过读取该cookie来获取变量的值

3.将查看的Internet页存储在cookies临时文件夹内，能提高以后的访问速度

三：setcookie()创建cookie

setcookie('name','jewel');    //至少传递一个参数'name'，这里有两个参数，'name'和name对应的值jewel

创建一个包含过期时间的cookie（当前时间戳加上一个时间 ，以秒为单位）

setcookie('name','jewel',time()+(10));    //这里设置cookie的过期时间为创建该cookie文件之后的10秒钟

cookie过期时间：如果不设置的话，浏览器关闭之后会自动删除这个cookie文件，如果设置了过期时间，在这个时间之内就算是重启计算机也不会删除该文件。不设置过期时间的cookie的生命周期为浏览器的会话时间，成为会话cookie，一般不保存在硬盘中，而是在内存中。设置了

四：超级全局变量$_COOKIE[]读取浏览器端的cookie值

五；isset()函数检测cookie是否存在

六：删除cookie

1.setcookie()函数删除cookie，只需将第二个参数设为空，第三个参数设为当前时间之前的时间，如现在的前一秒

setcookie('name','',time()-1);    //得到过去的时间，删除cookie

2.用浏览器手动删除

七：注意事项

1.在应用setcookie()函数创建cookie之前，不能有任何的HTML输出，即使是空格，空行也不行。因为cookie是经由服务器的程序通过http请求和响应发送到浏览器的，是http头标的组成部分，而头标必须在页面的其他内容发送之前发送

2.不要用cookie保存数据集或者其他大量的数据，也不是所有浏览器都支持cookie，也不要保存很隐私的数据。

3.cookie的值改变之后要刷新两遍才能显示新内容，也可以说他慢半拍。因为setcookie()函数执行之后，会向客户端发送一个cookie，如果不刷新或者浏览下一个页面，客户端不能将cookie送回

4.一个浏览器创建的cookie为30个，每个小于4kb，每个web站点的cookie个数最多20个

php学习--日期与时间

应用场合：表单提交时间，用户登录时间，数据库中数据的更新和删除的时间等

总结：

time()：获取当前时间戳（但是这个比较麻烦，参数含义不明确，一般不使用这个）

getdate()：将时间戳转化为人们可以看得懂的时间，数组形式

strtotime()将人们可读的日期转化为时间戳

mktime()将时间转化成时间戳

getlastmod()返回当前文件最后修改时间的时间戳

checkdate()：检验日期是否合法，boolean形式

date()：格式化本地时间或者日期，可以打印当前时间，date(‘Y-m-d h:i:sa’)

gettimeofday()获取当前时间，数组形式

localtime()返回本地时间（不常用）

date_default_timezone_get()获取默认时区

date_default_timezone_set()修改时区

一：checkdate()函数检测日期是否合法，布尔型

二：date()格式化一个本地时间或日期

格式：string date ( string $format [, int $timestamp ] )

date()的两个参数分别表示一个日期（必选）和一个时间戳（不写的话默认为Unix时间戳）。

date()表示的年月日时分秒

规则：

Y表示四位数的年份，y表示两位数的年份
M表示英文的月份缩写，m表示阿拉伯数字的月份
D表示英文的日缩写，d表示阿拉伯的日

H表示24小时的小时，h表示12小时的计时法
i表示分钟
s表示秒

a是小写的上午和下午

date()的第一个参数可以放一些format目录里不存在的符号，他们不会被解析，看起来排版清晰

打印目前的时间

echo date('Y-m-d h:i:sa');    //返回的是Unix时间戳的时间2016-11-13 08:34:23am，是东八区的时间，可能和实际时间不相同。

时间戳的概念：时间戳是文件属性中的创建，修改和访问时间。

原理：通过其他加密法将时间的数值转化为加密的数值，加密位数会发生改变安

好处：防止数值窃取之后被非法利用，起到了加密的作用

Unix时间戳：从1970年一月一日零时零分零秒到当前时刻，他是一个时间差，与时区没有关系

扩展： microtime() 返回当前 Unix 时间戳和微秒数，也就是gettimeofday()的前两个

作用：计算脚本运行时间（指的是很小很小的时间的差）

function fn(){
    list($a,$b) = explode(' ', microtime());    //以空格分割字符串，将分割后的字符串分别赋值给$a,$b
    return $a+$b;
}
$start = fn();
for ($i=0; $i<100000000 ; $i++) {     //时间越大，运行时间越大
    # code...
}
$end = fn();

echo round($end - $start,4);

运行此脚本的时间是3.1149秒

控制器：数据和视图之间的桥梁，数据就是通过控制器将数据发送到视图页面的

创建控制器的方法：

1.手动创建

控制器的位置：app->Http->Controllers下，我们就是在那里创建有意义的控制器

第一步：在controller下新建一个文件，取一个有意义的名字，例如：InputController

第二步：填写内容

<?php 
namespace App\Http\Controllers;    //指定命名空间，在它下边

class InputController extends Controller
{
    public function input()    //创建一个函数
    {
        echo "这是我手动创建的控制器";
    }

}
// 注意：laravel里所有用浏览器访问的函数都需要到routes里配置路径
// 所以要回到路由器配置一下路径app,http,routes

第三步：去routes里配置，用路由取访问控制器

// 定义指向控制器路由的动作，user/{id}是索要访问的页面，UserCotroller是你定义的控制器的名称，showProfile是UserCotroller里边的方法
// Route::get('user/{id}', 'UserController@showProfile')
Route::get('test','InputController@input');

第四步：输入localhost:8000/test，显示这是我手动创建的控制器

开启laravel的方法：cd到Laravel目录，命令行输入 php artisan serve

PHP Warning: Module ‘openssl’ already loaded in Unknown on line 0的解决方法：php.ini中搜索openssl，有两个相同的 ;extension=php_openssl.dll，将其中一个的分号去掉，一个保留

里的去掉分号，另一个保留

php artisan make:composer不能找到命名空间。composer语法错误，有可能是conposer.json文件有注释修改等不合语法的操作

2.命令行创建

第一步：在laravel的目录下打开命令行，快捷键是进入目录，shift+右键打开

第二步：输入：php artisan make:controller 你的控制器名字，比如php artisan make:controller HelloController,就可以创建一个HelloController的控制器，配置如下

<?php

namespace App\Http\Controllers;

    }

    //sort的第二个可选参数
    $number = array(2,12);
    sort($number,SORT_STRING);    //第二个参数为sort_string，说明要求数组元素按照字符串的顺序比较（无论数组元素是字符串还是数字，都会转化成字符串比较）

##1.定义

$students = array(
    array('aaa',18,'man'), //第0行
    array('bbb',19,'woman'), //第1行
    array('ccc',17,'man')    //第2行
    );
echo count($students);    //打印外数组长度的方法，即student的长度，是里边的三个数组，结果是3，这个是外层的数组student的个数。
// count($students);通俗地讲是二维数组的行数
echo count($students[1]);    //3
echo count($students[2]);    //3
echo count($students[0]);    //打印内数组的长度的方法，即student的第1个元素的长度，结果是3

4.打印数组元素的方法

没有键的情况

$students = array(
array('aaa',18,'man',), //第0行
array('bbb',19,'woman','green'), //第1行
array('ccc',17,'man','red')    //第二行
);
// 这两种方法适合下标为0,1,2...的情况
//打印二维数组的第一种方法，此方法很麻烦
echo $students[0][0]."|".$students[0][1]."|".$students[0][2]."<br>";
echo $students[1][0]."|".$students[1][1]."|".$students[1][2]."<br>";
echo $students[2][0]."|".$students[2][1]."|".$students[2][2]."<br>";

//打印数组的第二种方法：for循环,两重循环打印二维数组
for ($i=0; $i < count($students); $i++) { 
    // echo $students;    //此时打印3个array，这是求student含有几个数组
    for ($j=0; $j < count($students[$i]); $j++) {     //这里用$i的原因：就算内层数组里元素的个数各不相同也可以打印的出来
        echo $students[$i][$j]."|";
    }
    echo "<br>";    //外层循环3次，每次加1个换行
}

php几种输出方式比较

  •   echo: 函数，无返回值（返回void），作用是打印字符串，还可以打印函数的类型和数组类型

  •   printf: 函数，返回字符串的长度，作用是打印字符串
  •   sprintf: 函数，返回字符串将字符串保存在内存中（不在浏览器下显示）

  •   print_r: 函数，用来打印数组中的所有元素
